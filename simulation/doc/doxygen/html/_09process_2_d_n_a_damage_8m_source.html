00005 % @description
00006 %   Biology
00007 %   ======================
00008 %   Simulates a spectrum of DNA lesions repaired by base excision repair,
00009 %   nucleotide excision repair, and homologous recombination repair:
00010 %   - DNA strand breaks: caused by
00011 %     - hydrolysis and beta-elimination of abasic sites
00012 %     - ionizing radiation (alpha, beta particles, protons, electrons,
00013 %       gamma rays)
00014 %   - Spontaneous base loss
00015 %   - Base damage: caused by
00016 %     - Spontaneous deamination
00017 %     - oxidation -- oxygen radicals generated by metabolism, ionizing radiation, chemicals
00018 %     - alkylation -- carbon radicals generated by metabolism, ionizing radiation, chemicals
00019 %     - amination -- nitrogen radicals generated by metabolism, ionizing radiation, chemicals
00020 %   - Intrastrand cross links: caused by
00021 %     - UV-B induced cyclization
00022 %
00023 %   The simulation does not consider the following types of DNA damage
00024 %   - interstrand cross links: caused by bifunctional akylating agents
00025 %   - psoralens
00026 %   - DNA-protein cross links
00027 %   - Sugar-phosphate damage other than strand breaks
00028 %
00029 %   Additionally, the simulation does not consider mutiple damages per
00030 %   base or sugar-phosphate.
00031 %
00032 %   Categorizes strand breaks in DNA segments (216 bases) according to
00033 %   scheme described in the track structure model literature [PUB_0486]:
00034 %   - NB     -- no strand break
00035 %   - SSB    -- single strand break
00036 %   - SSB+   -- multiple single strand breaks on segment all on same strand
00037 %   - 2SSB   -- single strand breaks on both strand separated by at least 10 bases on DNA segment
00038 %   - DSB    -- two single strand breaks on opposite strands within 10 bases
00039 %   - DSB+   -- double strand break with additional strand break on DNA segment
00040 %   - DSB++  -- two single strand breaks on same DNA segment
00041 %
00042 %   Effect of DNA damage on physiology
00043 %   ++++++++++++++++++++++
00044 %   The exact effect of DNA damage on physiology is not well characterized.
00045 %   Certain kinds of damage are believed to arrest transcription
00046 %   [PUB_0780]. Tornaletti and Hanawalt provide the most extensive review
00047 %   of the effect of specific damages on transcription [PUB_0781, PUB_0782,
00048 %   PUB_0783]. For simplicity we we model all damages (except m6AD involved
00049 %   in R/M systems) as barriers to protein binding and function. This is
00050 %   implemented with the chromosome state class.
00051 %

00109 %   Representation
00110 %   ======================
00111 %   7 properties are used to keep track of the damage state of DNA. The first
00112 %   dimension of each has length equal to that of the chromosome.
00113 %   - gapSites                  true/false    sugar gaps
00114 %   - abasicSites               true/false    abasic sites
00115 %   - damagedSugarPhosphates    enumeration   damaged sugar phosphates
00116 %   - damagedBases              enumeration   damaged bases
00117 %   - intrastrandCrossLinks     true/false    intrastand cross link
00118 %   - strandBreaks              true/false    strand break (missing sugar-phosphate bond)
00119 %   - hollidayJunctions         true/false    holliday junction
00120 %
00121 %   substrates represents the amounts of free metabolites. initializeConstants
00122 %   synchronizes the substrates of this process with that of the DNA repair
00123 %   process to make it easy for these processes to communicate. That is
00124 %   initializeConstants sets the substrates of both processes equal to the union
00125 %   of the two, are orders the substrates the same in both processes. Thus
00126 %   substrate indices defined in process are valid on properties received from
00127 %   the other process, and it is unecessary to remap properties in
00128 %   copyFromState.
00129 %
00130 %   stimuli represents the amounts of DNA damaging agents including several
00131 %   kinds of radiation.
00132 %
00133 %   This process uses three properties rather than reactionStoichiometryMatrix
00134 %   to represent DNA damaging reactions and the DNA, damaging agents, and free
00135 %   metabolites that participate in them. These three matrices are a
00136 %   factorization of the super class reactionStoichiometryMatrix property.
00137 %   - reactionSmallMoleculeStoichiometryMatrix: the stochiometry of free
00138 %     metabolites in damage reactions
00139 %   - reactionDNAStoichiometryMatrix: the stoichiometry of DNA bases in
00140 %     damage reactions
00141 %   - reactionRadiationStoichiometryMatrix: the stoichiometry of radiation
00142 %     stimuli in damage reactions
00143 %
00144 %   This process uses reactionVulnerableMotifs, rather the super class
00145 %   reactionModificationMatrix property to represent the affects of the DNA
00146 %   damage reaction on macromolecules. reactionVulnerableMotifs represents the DNA
00147 %   sequence(s) vulnerable to each damage reaction.
00148 %
00149 %   Initialization
00150 %   ======================
00151 %   The DNA is initialized to the undamaged state, except that MunI
00152 %   restriction/modification sites are initialized to the methylated state (see
00153 %   DNA Repair process).
00154 %
00155 %   Simulation
00156 %   ======================
00157 %   For each reaction
00158 %     1. Compute rate that individual vulnerable DNA regions succombs to damage
00159 %     2. Randomly select among vulnerable DNA regions at rate computed in step
00160 %        (1)
00161 %     3. Mark DNA selected in step (2) as damaged by updating gapSites,
00162 %        abasicSites, damagedSugarPhosphates, damagedBases,
00163 %        intrastrandCrossLinks, strandBreaks, and hollidayJunctions
00164 %     4. Decrement/increment free metabolites reaction reactants and products
00165 %        using reactionSmallMoleculeStoichiometryMatrix
00166 %   end

00536             %loop over reactions in random order

00538             for i = 1:length(this.reactionWholeCellModelIDs)

00540                 
00541                 %maximum number of reactions, based on substrate availability
00542                 maxReactions = floor(min(this.substrates ./ max(0, -this.reactionSmallMoleculeStoichiometryMatrix(:, j))));
00543                 if maxReactions <= 0
00544                     continue;
00545                 end
00546                 
00547                 % probability that valid site undergoes reaction
00548                 radiationLclIdx  = this.reactionRadiation(j);
00549                 if radiationLclIdx ~= 0
00550                     selectionProbability = this.stepSizeSec * this.reactionBounds(j, 2) * this.substrates(radiationLclIdx);
00551                 else
00552                     selectionProbability = this.stepSizeSec * this.reactionBounds(j, 2);
00553                 end
00554                 if selectionProbability == 0
00555                     continue;
00556                 end
00557                 
00558                 %randomly damage sites
00559                 [positionsStrands, sideEffects] = this.chromosome.setSiteDamaged(...
00560                     this.reactionDamageTypes{j}, this.reactionDNAProduct(j), selectionProbability, ...
00561                     maxReactions, this.reactionVulnerableMotifs{j}, this.reactionVulnerableMotifTypes{j});
00562                 if isempty(positionsStrands)
00563                     continue;
00564                 end
00565                 
00566                 % use small molecule reactants, produce small molecule products
00567                 this.substrates = this.substrates + ...
00568                     size(positionsStrands, 1) * this.reactionSmallMoleculeStoichiometryMatrix(:,j);
00569                 
00570                 %side effects
00571                 if ~isempty(sideEffects)
00572                     this.simulationStateSideEffects = [this.simulationStateSideEffects; sideEffects];
