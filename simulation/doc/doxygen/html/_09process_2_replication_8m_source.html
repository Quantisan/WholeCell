00003 % @wholeCellModelID Process_Replication
00004 % @name             Replication
00005 % @description
00006 %   Biology
00007 %   =================
00008 %   DNA replication by replisomes is initiated by DnaA complex formation near
00009 %   the oriC and proceeds bidirectionally from oriC to terC in the 5'->3'
00010 %   direction along leading strand as well as in Okazaki fragments in the 5'->3'
00011 %   direction along lagging strands. This process models the locations, kinetics,
00012 %   and biochemistry of all the replication proteins:
00013 %   - replicative DNA helicase
00014 %   - DNA primase
00015 %   - DNA polymerase core
00016 %   - Beta-clamp loading complex / gamma complex
00017 %   - Beta-clamp
00018 %   - DNA ligase
00019 %   - single stranded binding proteins
00020 %
00021 %   The exact mechanism replication initiation in M. genitalium is unknown.
00022 %   Furthermore, because M. genitalium does not contain a DnaC homolog it is
00023 %   difficult to infer M. genitalium repliciation initiation from studies of
00024 %   other bacterial species.
00025 %
00026 %   Knowledge Base
00027 %   =================
00028 %   The knowledge base contains data curated from the literature and databases:
00029 %   - the chromosome DNA sequence
00030 %   - the DNA footprints of all replication proteins
00031 %   - DNA binding protein displacement reactions (that is which proteins can
00032 %     displace which other proteins from the chromosome)
00033 %   - values of various structural and kinetic parameters
00034 %   - subunit composition of the replication proteins
00035 %
00036 %   Representation
00037 %   =================
00038 %   The properties substrates, enzymes, and boundEnzymes represent the counts of
00039 %   free metabolites, free replication proteins, and chromosally-bound
00040 %   replication proteins. The chromosomes property polymerizedRegions represents
00041 %   the regions of the chromosomes which have been polymerized (and the
00042 %   base-pairing of strands). The chromosomes property strandBreaks represents
00043 %   strand breaks 5' to each base. The chromosomes property complexBoundSites
00044 %   represents represents the specific chromosomal location of all chromosomally
00045 %   bound proteins.
00046 %
00047 %   Spatial Model
00048 %   +++++++++++++++++
00049 %   - Helicase is centered on the boundary between ssDNA and dsDNA. The position
00050 %     over which it is centered is the next position to be melted.
00051 %   - Polymerase core is centered on the boundary between ssDNA and dsDNA. The
00052 %     position over which it is centered is the next position to be polymerized.
00053 %   - There is no gap between the helicase and polymerase core or between the
00054 %     polymerase core and the beta clamp.
00055 %   - Backup beta clamps bind slightly upstream of the start site of Okazaki
00056 %     fragments such that there will be no gap between the polymerase and beta
00057 %     clamp, and the polymerase core will be centered on the Okazaki fragment
00058 %     start site
00059 %   - At replication initiation a the mother strands are separated such that the
00060 %     leading polymerase cores are centered at oriC+-1 and the helicases are
00061 %     slightly (11 nt) ahead
00062 %   - During replication initiation (and the final step of replication after the
00063 %     last Okazaki fragment has completed) the lagging polymerase and primase
00064 %     are accounted for as part of a complex on the leading strand (containing
00065 %     also the helicase, leading polymerase, gamma complex, and leading beta
00066 %     clamp). At all other times, the lagging polymerase, lagging beta clamp,
00067 %     and primase are accounted for as a complex on a different strand. This
00068 %     allows us to separately keep track of the leading and lagging polymerase
00069 %     positions.
00070 %   - Backup beta clamps can't binding until half of the previous Okazaki
00071 %     fragment has been polymerized
00072 %   - In general there will be approximately 1 Okazaki fragment length gap
00073 %     between the progess of DNA polymerase on the leading and lagging strands
00074 %
00075 %   Initialization
00076 %   =================
00077 %   The mother chromosome is initialized by the Chromosome class and
00078 %   several processes:
00079 %   - completely synthesized
00080 %   - mother strands base pairing
00081 %   - undamged, except methylated at restriction/modification sites (DNA repair
00082 %     process)
00083 %   - bound by various proteins
00084 %     - SMCs (DNA condensation)
00085 %     - RNA polymerase (Transcription)
00086 %     - Transcription factors (Transcriptional regulation)
00087 %     - Topoisomerases (supercoiling)
00088 %     - DnaA (replication initiation)
00089 %
00090 %   Simulation
00091 %   =================
00092 %   The simulation consists of 8 subfunctions executed in a random order:
00093 %   - Initiate replication (initiateReplication)
00094 %     If DnaA complex assembled at OriC and sufficient protein and metabolites,
00095 %     unwind small segment of DNA and binding helicase, primase, polymerase,
00096 %     gamma complex, and beta clamp to chromosomes. Associate all proteins with
00097 %     the leading strand. Chromosome will take care of dissassembly of the
00098 %     DnaA complex. ReplicationInitiation will take care of dissociating the
00099 %     released DnaA-ATP polymers.
00100 %
00101 %   - Advance replisomes: unwind and polymerize DNA, release SSBs (unwindAndPolymerizeDNA)
00102 %     1. If first Okazaki fragment starting, associate primase, lagging polymerase
00103 %        with lagging strand.
00104 %     2. Advance leading and lagging polymerases and helicases up to limits
00105 %        1. Polymerase and primase kinetics
00106 %        2. Available dNTP for polymerization and energy for unwinding
00107 %        3. Prevent leading strand progressing if no SSBs bound to lagging
00108 %           strand
00109 %        4. Accessibility of upstream regions to helicase and polymerases
00110 %        5. Don't polymerize leading strand past terC or lagging strands past
00111 %           ends of Okazaki fragments
00112 %        6. Don't allow the leading strand and helicase to go way beyond the
00113 %           lagging strand progress.
00114 %        7. Pause progress is RNA polymerase is encountered.
00115 %           If the replication loop (helicase) hits an RNAP, polymerization
00116 %           pauses, the RNAP falls off, and its transcript is degraded. If it's
00117 %           a head-on collision, the replication loop will not proceed for some
00118 %           unknown amount of time (a fittable parameter). If it's a codirec-
00119 %           tional collision, polymerization will continue at full speed the
00120 %           following time step. (Mirkin 2004, Mirkin 2006, Mirkin 2007).
00121 %           If occupied by RNA-polymerase, then calculate "occupied DNA" by base
00122 %           polymerase is currently on -9 through +2. (Neidhardt 1990).
00123 %
00124 %   - Bind SSBs (freeAndBindSSBs)
00125 %     Bind free SSB 4mers to stochastically to deterministically selected
00126 %     positions within single-stranded regions as SSB 8mers
00127 %
00128 %   - Dissociate free SSB 8mers into 2 SSB 4mers (dissociateFreeSSBComplexes)
00129 %     Dissociate released SSB 8mers into 2 SSB 4mers
00130 %
00131 %   - Initiate Okazaki fragment by bind beta-clamp (initiateOkazakiFragment)
00132 %     Binding beta-clamp just downstream of the start position of the next
00133 %     Okazaki fragment. Requires that
00134 %     - There is beta-clamp monomer to form new beta-clamp dimer on chromosome
00135 %     - There is energy for the gamma-complex to form the new beta-clamp
00136 %     - The position is accessible to the beta-clamp
00137 %     - The leading helicase has already passed the position
00138 %     - The lagging strand is at least 1/2 done with the current Okazaki
00139 %       fragment
00140 %
00141 %   - Terminate Okazaki fragment by releasing beta-clamp (terminateOkazakiFragment)
00142 %     Release beta-clamp and associate lagging primase and polymerase with
00143 %     backup beta-clamp (or if terminating the last Okazaki fragment,
00144 %     associate the lagging primase and with the leading strand machinery).
00145 %     Mark end of Okazaki fragment as having a single strand break to be ligated
00146 %     by ligase.
00147 %     Occurs if:
00148 %     - Okazaki fragment finished polymerizing
00149 %     - SSBs bound to lagging strand
00150 %     - Lagging backup beta-clamp bound
00151 %     - Leading machinery has advanced beyond the Okazaki fragment start site
00152 %
00153 %   - Terminate replication (terminateReplication)
00154 %     Release bound replisomes from leading strands. Mark terC as having single
00155 %     strand breaks to be ligated by ligase. Occurs if:
00156 %     - Polymerization completed for both leading and lagging strands
00157 %     - Lagging strand ligated (except for last ligation at terC which will
00158 %       occur after the execution of this subfunction)
00159 %
00160 %   - Ligate DNA (ligateDNA)
00161 %     Stochastically ligate single strand breaks up to
00162 %     - Ligase kinetics
00163 %     - Ligase availability
00164 %     - NAD availability
00165 %
