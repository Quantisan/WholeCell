00006 %   Biology
00007 %   ===============
00008 %   Transcription is the first step in the synthesis of functional gene
00009 %   products where RNA polymerase and several accessory enzymes translate
00010 %   transcription units, or regions of the DNA containing 1 or more genes, into
00011 %   RNA molecules. Following transcription the RNA molecules follow one of
00012 %   two pathways:
00013 %   - mRNAs are used as templates for translation (see translation process)
00014 %   - r/s/tRNAs which are transcribed as molecules containing multiple genes are
00015 %     cleavaged into their individual genes (see RNA processing process), are
00016 %     modified at several bases (see RNA modification process) to improve their
00017 %     stability and enhance their catalytic activity, and finally act as
00018 %     ribozymes (r/sRNAs) or as adaptors between mRNAs and the amino acids they
00019 %     code for (tRNAs).
00020 %
00021 %   Transcription begins with the recruitment of RNA polymerase to a promoter
00022 %   with the help of the sigma initiation factor and possiblity transcription
00023 %   factors. Next elongation factors are recruited, RNA begins to be
00024 %   polymerized, and sigma factor is released. Finally the RNA polymerase
00025 %   reaches  a terminator at the the end of the transcription unit, and
00026 %   with the help of termination factors releases the polymerized RNA and
00027 %   dissociates from the DNA. Termination in E. coli occurs via either a
00028 %   Rho-dependent (50%) or Rho-independent mechanism (50%). Rho-dependent
00029 %   termination is catalyzed by the hexameric ATP-dependent helicase Rho.
00030 %   Rho is not essential in B. subtilis [PUB_0234]. Rho-independent
00031 %   termination occur via the intrinsic properties RNA which disrupt RNA
00032 %   polymerase-DNA binding. Terminator hairpins are not predicted in M.
00033 %   genitalium. In E. coli termination is incompetition with
00034 %   antitermination. However antitermination has not been reported in any
00035 %   mcyoplasma [PUB_0182].
00036 %
00037 %   As soon as the RNA begins to polymerized, even prior to termination, the
00038 %   mRNA transcripts may be bound by ribosomes and polymerized. For simplicity,
00039 %   our model doesn't represent this phenomenon.
00040 %
00041 %   Knowledge Base
00042 %   ===============
00043 %   The transcription unit structure was compiled from several sources:
00044 %   - Primary reports of cotranscribed genes [PUB_0176, PUB_0182, PUB_0186,
00045 %     PUB_0188, PUB_0188, PUB_0244, PUB_0247, PUB_0248, PUB_0249, PUB_0251]
00046 %   - OperonDB database of cotranscribed genes [PUB_0250]
00047 %   - Conservation of gene order across multiple species
00048 %   - Related function of adjacent genes
00049 %   - Expression levels as measured by microarrays of adjacent genes [PUB_0569]
00050 %   - Strandedness of adjacent genes
00051 %   - Weiner, Hermann, and Browning computational model of promoters and
00052 %     transcription unit start sites [PUB_0411]
00053 %
00054 %   The transcription unit structure is organized in the knowledge base, and is
00055 %   loaded into this process by the initializeConstants method.
00056 %
00057 %   The knowledge base also contains the measured expression and half-lives of
00058 %   many transcripts. These values are loaded by initializeConstants and fit by
00059 %   simulation.fitConstants to be consistent with other processes.
00060 %
00061 %   Representation
00062 %   ===============
00063 %   substrates, enzymes, boundEnzymes, and RNAs represent the counts of
00064 %   metabolites, free transcription enzymes, transcription enzymes bound to RNAs
00065 %   and RNA polymerases, and nascent RNAs.
00066 %
00067 %   rnaPolymerases.states represents the current state / pseudostate (actively
00068 %   transcribing, specifically bound, non-specifically bound, free,
00069 %   non-existent) of each RNA polymerase, where each state is indicated by the
00070 %   enumeration:
00071 %   - rnaPolymeraseActivelyTranscribingValue
00072 %   - rnaPolymeraseSpecificallyBoundValue
00073 %   - rnaPolymeraseNonSpecificallyBoundValue
00074 %   - rnaPolymeraseFreeValue
00075 %   - rnaPolymeraseNotExistValue (state exists as a way to account for memory
00076 %     allocated for future RNA polymerases)
00077 %   For actively transcribing polymerases rnaPolymerases.states also represents
00078 %   the position of the polymerase along the transcription unit.
00079 %
00080 %   That is entries of rnaPolymerases.states with the following values
00081 %   corresponding to these states:
00082 %     >= RNAPolymerases.activelyTranscribingValue: RNA polymerases position on genome actively transcribing
00083 %     == RNAPolymerases.specificallyBoundValue:    RNA polymerase specifically bound
00084 %     == RNAPolymerases.nonSpecificallyBoundValue: RNA polymerase non-specifically bound
00085 %     == RNAPolymerases.freeValue:                 RNA polymerase free
00086 %     == RNAPolymerases.notExistValue:             RNA polymerase doesn't exist
00087 %
00088 %   transcripts.boundTranscriptionUnits represents the particular transcription
00089 %   unit to which each actively transcribing and specifically bound polymerase
00090 %   is bound.
00091 %
00092 %   rnaPolymeraseStateExpectations represents the expected occupancies of the
00093 %   RNA polymerase states.
00094 %
00095 %   transcriptionUnitBindingProbabilities represents the relative affinity of
00096 %   RNA polymerases for the promoters of each transcription unit.
00097 %   transcriptionFactorBindingProbFoldChange represents the fold change affect of
00098 %   transcription factors on the relative affinities of the RNA polymerse for
00099 %   the promoters. RNA polymerases are assigned to
00100 %   transcription units weighted by the product of
00101 %   transcriptionUnitBindingProbabilities and
00102 %   transcriptionFactorBindingProbFoldChange.
00103 %
00104 %   Initialization
00105 %   ===============
00106 %   All RNAs are initialized to the mature state. This is implemented by the
00107 %   simulation class initializeState method.
00108 %
00109 %   RNA polymerases are initialized to their steady state:
00110 %   - Each RNA polymerase is randomly assigned (with replacement) to one of the
00111 %     actively transcribing, specifically bound, non-specifically bound, or free
00112 %     states weighted by the expected occupancy of each state
00113 %     (rnaPolymeraseStateExpectations)
00114 %   - Actively transcribing and specifically bound polymerases randomly assigned
00115 %     to transcription units weighted by their transcription rates
00116 %     (transcriptionUnitBindingProbabilities)
00117 %   - Each transcription unit to which an actively transcribing polymerase has
00118 %     been assigned is divided into 1 segment for each polymerase
00119 %   - Actively transcribing polymerases randomly assigned to positions within
00120 %     the assigned segment of their assigned transcription unit (positions near
00121 %     the segment border are not allowed to prevent polymerases from being too
00122 %     close to each other) with uniform probably.
00123 %
00124 %   Simulation
00125 %   ===============
00126 %   Evolves the state of RNA polymerase using a markov chain model with four
00127 %   states:
00128 %   - actively translating
00129 %   - specifically bound
00130 %   - non-specifically bound
00131 %   - free
00132 %
00133 %   Transition probabilities are designed to maintain the occupancy of each
00134 %   state within a narrow window around their expected values. Transition
00135 %   probability are determined by four logistic control functions. These can
00136 %   be tuned with the constants
00137 %   - rnaPolymeraseStateExpectations
00138 %
00139 %   RNA polymerase are created in the free state.
00140 %
00141 %   Actively transcribing state:
00142 %   1. Release sigma factor if after first second of elongation
00143 %   2. Elongate transcript according to nucleic acid limits (substrates)
00144 %      if elongation factors are available
00145 %   3. If transcription complete and termination factor available
00146 %      - release transcript
00147 %      - transition RNA polymerase to free state
00148 %      - increment gene expression
00149 %      Otherwise remain in active state
00150 %
00151 %   Specifically bound State:
00152 %   - Can transition to active, specifically bound, non-specifically bound,
00153 %     or free states
00154 %   - Transition into state only if a free sigma factor is available
00155 %     1. Decrement number of free sigma factors
00156 %     2. Pick a transcription unit (tu) to bind to according to
00157 %     Expression transcription unit i~prob(ribosome releases tu i|ribosome active)
00158 %                      =prob(ribosome within RNA polymerase elongation rate bases of length of tu i|ribosome active)
00159 %                      =prob(ribosome within RNA polymerase elongation rate bases of length of tu i|ribosome active, bound to tu i)*prob(ribosome bound to tu i|ribosome active)
00160 %                      =[(RNA polymerase transcription rate)/(length of tu i)] * [(length of tu i)*prob(binding tu i|binding)]
00161 %     prob(binding tu i | binding)~expression tu i
00162 %
00163 %   Non-specifically bound state:
00164 %   - Can transition to specifically bound, non-specifically bound, or free
00165 %     states
00166 %
00167 %   Free state:
00168 %   - Can transition to specifically bound, non-specifically bound, or free
00169 %     states
00170 %
00171 %   Algorithm
00172 %   +++++++++++++++
00173 %   1. Randomly transition RNA polymerases among activlely transcribing,
00174 %      specifically bound, non-specifically, bound, and free states weighted by
00175 %      state transition probabilities. Update rnaPolymerases.states.
00176 %   2. Randomly assign RNA polymerases entering the specifically bound to
00177 %      specific transcription units weighted by the product of
00178 %      transcriptionUnitBindingProbabilities and
00179 %      transcriptionFactorBindingProbFoldChange. Update
00180 %      transcripts.boundTranscriptionUnits.
00181 %   3. Assign RNA polymerase entering the actively transcribing state sigma
00182 %      factors. Update enzymes and boundEnzymes.
00183 %   4. Simulate RNA polymerization by actively transcribing RNA polymerases with
00184 %      the aid of elongation factors. Allocate available nucleic acids among the
00185 %      actively transcribing RNA polymerases. Release sigma factors from RNA
00186 %      polymerases that started at the beginning of the transcription unit and
00187 %      progressed. Update rnaPolymerases.states. Update substrates. Update enzymes
00188 %      and boundEnzymes.
00189 %   5. If termination factors are available dissociate RNA polymerases which
00190 %      have reached the terminus of the transcription they're bound to, and
00191 %      release RNAs. Update rnaPolymerases.states and
00192 %      transcripts.boundTranscriptionUnits. Increment RNAs.
