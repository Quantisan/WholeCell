00001 %Translation
00002 %
00003 % @wholeCellModelID Process_Translation
00004 % @name             Translation
00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   Translation is the first step in the synthesis of function proteins whereby
00009 %   the ribosome, accessory enzymes, and tRNAs transcode mRNAs and produce
00010 %   amino acid polymers. Following translation proteins must be matured to
00011 %   become fully functional:
00012 %    1. Protein Processing I: N-terminal methionine deformylation and cleavage
00013 %    2. Protein Translocation: Translocation of integral membrane proteins,
00014 %       lipoproteins, and secreted proteins into and through the cell membrane
00015 %    3. Protein Processing II: Diacylglyceryl addition to lipoproteins. Signal
00016 %       sequence cleavage of lipo- and secreted proteins.
00017 %    4. Protein Folding: Folding and prosthetic group coordination of proteins.
00018 %    5. Protein Modification: Addition of covalently attached chemical groups to
00019 %       proteins.
00020 %    6. Macromolecular Complexation: Incorporation of protein monomers into
00021 %       larger macromolecular assemblies.
00022 %    7. Ribosome Assembly: Special case of macromolecular complexation of the
00023 %       ribosome which requires GTPases.
00024 %    8. Terminal Organelle Assembly: Special case of translocation which
00025 %       sequesters several protein species in a membrane-bound bleb referred to
00026 %       as the terminal organelle.
00027 %    9. Protein Activation: Functional activation and inactivation of proteins
00028 %       in response to their surrounding chemical environment.
00029 %
00030 %   Translation begins with the recruitment of the 30S and 50S ribosomal
00031 %   particles and initiation factor 3 (IF3) to an mRNA molecule. Next the
00032 %   ribosomal particles assembly into a 70S ribosome on the mRNA molecule with
00033 %   the help of initiation factors. Third the ribosome polymerizes amino
00034 %   acids with the help of elongation factors. Finally release factor
00035 %   (MG_258) recognizes the stop codon UAG, release factor hydolyzes the
00036 %   peptidyl tRNA bond and dissociates, ribosome recycling factor
00037 %   dissociates the E-site tRna, elongation factor G release the release
00038 %   factor and 50S ribosome, and initiation factor 3 dissociates the 30S
00039 %   ribosome, P-site tRNA, and mRNA.
00040 %
00041 %   When a ribosome stalls its lastest tRNA is expelled and replaced by the
00042 %   tRNA-like domain of a tmRNA molecule. Next the mRNA-like domain of the tmRNA
00043 %   expels the bound mRNA. Third the ribosome resumes polymerization, now
00044 %   using the tmRNA's mRNA-like domain as its template. This results in the
00045 %   production of an amino acid polymer containing a C-terminal proteolysis tag.
00046 %   Finally, the proteolysis tag will be recognized by the protein degradation
00047 %   machinery, and the amino acid polymer will be degraded into its individual
00048 %   component amino acids.
00049 %
00050 %   This process simulates protein translation by ribosomes and acessory
00051 %   initiation, elongation, and termination factors. The process also simulates
00052 %   the identification of stalled ribosomes by the tmRNA, the replacement of the
00053 %   tRNA and mRNA with the tmRNA, and the synthesis of the proteolysis tag
00054 %   encoded by the tmRNA's mRNA-like domain.
00055 %
00056 %   Knowledge Base
00057 %   ===============
00058 %   The knowledge base contains the region of the genome each mRNA species
00059 %   corresponds to, and the location of the tmRNA mRNA-like domain on the
00060 %   genome. This information is converted into a sequence of tRNAs which are
00061 %   required to polymerize each protein and proteolysis tag by the knowledge
00062 %   base class' computeTRNASequences method.
00063 %
00064 %   Representation
00065 %   ===============
00066 %   The properties substrates, enzymes, monomers, mRNAs, freeTRNAs,
00067 %   aminoacylatedTRNAs, and aminoacylatedTMRNA represents the counts of
00068 %   metabolites, translation enzymes, nascent peptides, mature mRNAs, free
00069 %   tRNAs, aminoacylated tRNAs, and aminoacylated tmRNAs.
00070 %
00071 %   monomerTRNASequences and tmRNAProteolysisTagTRNASequence represent the
00072 %   sequences of tRNAs required to polymerize each protein and proteolysis tag.
00073 %   These tRNA sequences are computed from the DNA sequence of the mRNA gene /
00074 %   tmRNA gene by the knowledge  base classs' computeTRNASequences method.
00075 %
00076 %   ribosomeStates represents the state/pseudostate (actively translating, free,
00077 %   non-existent) of each ribosome. boundMRNAs indicates the mRNA
00078 %   species each actively translating ribosome is bound to. The non-existent
00079 %   pseudostate is used to keep track of elements in ribosomeStates
00080 %   and boundMRNAs which have been allocated.
00081 %   nascentMonomerLengths and proteolysisTagLengths represeent
00082 %   the position of actively  translating ribosomes on mRNAs, and if stalled on
00083 %   a tmRNA.
00084 %
00085 %   Initialization
00086 %   ===============
00087 %   All protein monomers are initialized to the mature state, and in their
00088 %   correct localization. This is achieved by the simulation class
00089 %   initializeState method.
00090 %
00091 %   Ribosomes are initialized to their steady state:
00092 %   - Each ribosome is randomly assigned (without replacement) to mRNA species,
00093 %     weighted by the current expression of the mRNAs,.
00094 %   - Each ribosome is randomly assigned to positions within the assigned mRNA
00095 %     with uniform probably.
00096 %   - No ribosomes are initialized to the stalled state, since the expected
00097 %     occupancy is negligible. No tmRNAs are initialized to the bound state.
00098 %
00099 %   tRNAs and tmRNAs are initialized to the aminoacyated state by
00100 %   simulation.initializeState.
00101 %
00102 %   Simulation
00103 %   ===============
00104 %   Evolves the states of ribosomes among two states:
00105 %   - actively translating
00106 %   - free
00107 %
00108 %   Transition to from the free state to the actively translating state is
00109 %   allowed when
00110 %   - ribosome binding factor A is present
00111 %   - initiation factors (IF-1, IF-2, and IF-3) are present
00112 %   - one unit of energy (GTP) is available.
00113 %   At this time the ribosome binds an mRNA randomly with uniform probability.
00114 %
00115 %   At the next iteration the initiation factors are released when the first
00116 %   amino acid, f-methionine binds and elongation begins, assuming that
00117 %   elongation factors (EF-tu, TS, and G) and energy (GTP) is available.
00118 %
00119 %   ASSUMPTION made here is that one of each is sufficient for each ribosome,
00120 %   but need a separate set for each ribosome. So each ribosome can only
00121 %   translate if a full set exists for it. Also, for cases in which
00122 %   translation of a peptide finishes partway through the time step, the EFs
00123 %   are free, but haven't added the complexity of another ribosome being able
00124 %   to take up EFs partway through the timestep.
00125 %
00126 %   Finally, once all amino acids of a protein have been translated,
00127 %   termination occurs if
00128 %   - at least one terminator (RF-1) available
00129 %   - at least one recycling factor available
00130 %   - at least one elongation factor G available
00131 %   - at least one trigger factor available
00132 %   - energy is available.
00133 %   The ribosome will be available to bind mRNA at the following iteration.
00134 %
00135 %   Ribosomes are created in the free state.
00136 %
00137 %   Algorithm
00138 %   +++++++++++++++
00139 %   1. Up to limit of elongation factors, randomly select actively translating
00140 %      ribosomes to elongate.
00141 %   2. Up to limit of initiation factors and energy, randomly select ribosomal
00142 %      particles to initiatate. These ribosome will be able to start elongating
00143 %      at the next time step. Randomly select mRNA species for each initiating
00144 %      mRNA to bind to, weighted by the counts of each mRNA species. Update
00145 %      ribosomeStates, boundMRNAs. Update substrates.
00146 %   3. Allocate available amino acids and energy among actively translating
00147 %      ribosomes. Update ribosome nascentMonomerLengths, or
00148 %      proteolysisTagLengths for stalled ribosomes with the number of
00149 %      polymerized bases. Update substrates.
00150 %   4. If ribosome has reached end of (t)mRNA and termination factors
00151 %      available, increase count of protein and dissolve (t)mRNA-ribosome
00152 %      complex. Update substrates.
00153 %   5. If ribosome hasn't advanced, then with a small probability transition
00154 %      ribosome to stalled state. Expel mRNA and replace with tmRNA. Update
00155 %      ribosomeStates. Update substrates.
00156 %
00157 %   References
00158 %   ===============
00159 %   1. Petry S, Weixlbaumer A, Ramakrishnan V (2008). The termination of
00160 %      translation. Curr Opin Struct Biol. 18(1):70-7. [PUB_0226].

00378         %              modified monomers --(complexation)---> nascent complexs
00379         % nascent complexs + metabolites --(folding)--------> mature complexes
00380         %          monomer/complex + H2O --(decay)----------> AAs + modified AAs

00382         %            lipoyl lysine + H2O --(catabolism)-----> lysine + lipoate
00383         function [bmProd, byProd, minEnzExp, maxEnzExp] = calcResourceRequirements_LifeCycle(this, ~, states)
00384             %% initialize
00385             bmProd = zeros(size(this.substrateWholeCellModelIDs));
00386             byProd = zeros(size(this.substrateWholeCellModelIDs));
00387             minEnzExp = zeros(size(this.enzymeWholeCellModelIDs));
00388             maxEnzExp = Inf(size(this.enzymeWholeCellModelIDs));
00389             
00390             %% substrate and byproducts
00391             
00392             %breaking tRNA-peptidyl bond 
00393             bmProd(this.substrateIndexs_water) = ...
00394                 bmProd(this.substrateIndexs_water) + ...
00395                 this.polypeptide.monomerLengths' * states.monomerProductions;
00396             byProd(this.substrateIndexs_hydrogen) = ...
00397                 byProd(this.substrateIndexs_hydrogen) + ...
00398                 this.polypeptide.monomerLengths' * states.monomerProductions;
00399             
00400             %peptide bond formation
00401             byProd(this.substrateIndexs_water) = ...
00402                 byProd(this.substrateIndexs_water) + ...
00403                 max(0, this.polypeptide.monomerLengths - 1)' * states.monomerProductions;
00404             
00405             %energy
00406             translationEnergy = ...
00407                 2 * this.polypeptide.monomerLengths' * states.monomerProductions + ...  %elongation
00408                 3 * sum(states.monomerProductions);                                     %initiation, termination
00409             bmProd(this.substrateIndexs_gtp)       = bmProd(this.substrateIndexs_gtp)       + translationEnergy;
00410             bmProd(this.substrateIndexs_water)     = bmProd(this.substrateIndexs_water)     + translationEnergy;
00411             byProd(this.substrateIndexs_gdp)       = byProd(this.substrateIndexs_gdp)       + translationEnergy;
00412             byProd(this.substrateIndexs_phosphate) = byProd(this.substrateIndexs_phosphate) + translationEnergy;
00413             byProd(this.substrateIndexs_hydrogen)  = byProd(this.substrateIndexs_hydrogen)  + translationEnergy;
00414             
00415             %% enzymes
00416 
00417             %Ribosome
00418             fractionInitiating = this.ribosomeElongationRate *  ...
00419                 states.monomerProductions0' * (1 ./ this.polypeptide.monomerLengths) / ...
00420                 sum(states.monomerProductions0); %fraction of ribosomes that are initiating
00421             aaPolymerized0 = (states.monomerProductions0' * this.polypeptide.monomerLengths);
00422             minEnzExp(this.enzymeIndexs_ribosome30S) = 1.65 * aaPolymerized0 / this.ribosomeElongationRate / (1 - fractionInitiating);

00428             minEnzExp(this.enzymeIndexs_terminationFactor)     = 3;
00429             minEnzExp(this.enzymeIndexs_recyclingFactor)       = 3;
00430             minEnzExp(this.enzymeIndexs_elongationGFactor)     = 3 + minEnzExp(this.enzymeIndexs_elongationGFactor);
00431 
00432             %abortive translation enzymes
00433             minEnzExp(this.enzymeIndexs_tmRNA)                 = 3;
00434             minEnzExp(this.enzymeIndexs_tmRNABindingProtein)   = 3;
00435             minEnzExp(this.enzymeIndexs_peptidylTRNAHydrolase) = 3;
00436         end
