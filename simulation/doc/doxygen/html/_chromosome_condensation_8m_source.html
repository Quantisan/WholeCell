00004 % @name             Chromosome Condensation
00005 % @description
00006 %   Biology
00007 %   =====================================================================
00008 %   Chromosome segregation requires that the DNA be highly compacted.
00009 %   Structural maintenance of chromosome (SMC) complexes are "V" shaped
00010 %   proteins (with an head and two legs) that induce positive supercoils in
00011 %   double stranded DNA (Porter et al., 2004). The complexes are believed to
00012 %   work with a lock and key mechanism in which first DNA is looped around
00013 %   the legs of the SMC complex, and then an ATP is bound between the two
00014 %   tails to lock the SMC complex in place. The complexes bind and clamp the
00015 %   DNA causing many loops in the DNA and compacting it. The loops around
00016 %   each leg occupy 90bp. A loop of about 450bp forms between the two SMC
00017 %   complex legs (Jensen and Shapiro, 2003, Strick and Kawaguchi, 2004).
00018 %   Further, it has been inferred that there is about 1 SMC complex per every
00019 %   7000bp (Jensen and Shapiro, 2003).
00020 %
00021 %   The initial chromosome is bound by SMC complexes averaging 7000bp spacing.
00022 %   As the replication loop proceeds, the SMC complexes that it encounters
00023 %   are displaced.  Once the DNA has been replicated, SMC complexes are
00024 %   randomly bound to the DNA such that their spacing averages 7000bp. Each
00025 %   SMC complex occupies 630bp, and no two SMC complexes can occupy the same
00026 %   space. Both the decision of whether an SMC complex will bind at a given
00027 %   time point and the binding location are random. SMC complexes do not fall
00028 %   off the chromosomes unless due to the force of the replication loop. The
00029 %   two chromosomes are tracked separately.
00030 %
00031 %   Knowledge Base
00032 %   =====================================================================
00033 %   The knowledge base contains the values of the parameters: smcSepNt and
00034 %   smcSepProbCenter. The knowledge base also contains the measured DNA
00035 %   footprint of each SMC complex. These footprints are loaded by the
00036 %   Chromosome class from the knowledge base. This class retrieves the
00037 %   footprint from the Chromosome class.
00038 %
00039 %   Representation
00040 %   =====================================================================
00041 %   substrates, enzymes, and boundEnzymes represent the counts of free
00042 %   metabolites, free SMC complexes, and chromosome-bound SMC complexes. The
00043 %   chromosomes property represents the specific base positions where the
00044 %   chromosome-bound SMC complexes are located.
00045 %
00046 %   enzymeDNAFootprints represents the experimentally measured DNA footprint of each
00047 %   SMC complex. smcSepNt represents the experimentally observed average SMC
00048 %   complex spacing  [PUB_0517]. smcSepProbCenter is a parameter which controls
00049 %   the SMC binding probability transfer function. smcSepProbCenter is not an
00050 %   experimentally measured quantity. Rather, its value is pinned by several
00051 %   constraints: implemented in the ChromosomeCondensation_Test class
00052 %   testCalculateBindingPositionWithinRegion and testInitializeStateConverged
00053 %   methods. These constraints are:
00054 %   - Consistent with an SMC density of approximately 1/smSepNt
00055 %   - Consistent with fast binding of multiple SMCs to large unbound regions
00056 %   - Consistent with slow binding of SMCs to small unbound regions
00057 %
00058 %   Algorithm
00059 %   =====================================================================
00060 %   1. Calculate expected number of SMC complexes that should bind chromosomes
00061 %      a. Calculate regions where SMCs can bind (regions either between SMC
00062 %         complexes or between SMC complexes and replication bubble)
00063 %      b. Compute the expected number of binding complexes as the ratio of the
00064 %         sum of the lengths of the regions to the average SMC spacing
00065 %   2. For 1 to minimum of free SMC complexs, ATP, and expected number of SMCs
00066 %      that that should bind chromosome
00067 %      A. Calculate regions where SMCs can bind (regions either between SMC
00068 %         complexes or between SMC complexes and replication bubble)
00069 %         a. Starting coordinate
00070 %         b. Chromosomes
00071 %         c. Lengths
00072 %         d. Probability of SMC binding each region
00073 %      B. Pick a region for SMC complex to bind
00074 %      C. Pick a position within region for SMC complex to bind
00075 %      D. Form SMC-ADP complex:
00076 %         a. Decrement SMC. Increment SMC-ADP
00077 %         b. Decrement ATP, H2O. Increment PI, H.
00078 %      E. Bind SMC-ADP complex to chromosome
00079 %         a. Decrement free SMC-ADP complex
00080 %         b. Increment bound SMC-ADP complex
00081 %
00082 %   The probability that an SMC complex binds a region of length L is given by
00083 %   the step function
00084 %     p(L) = 1/smcSepNt * max(0, L/2-smcSepProbCenter)
00085 %
00086 %   The conditional probability that an SMC complex binds a position within
00087 %   region assuming that the SMC complex is binding the region is given by
00088 %    p(x) = 1/(L-2*smcSepProbCenter)  if x>smcSepProbCenter and x<L-smcSepProbCenter,
00089 %           0                         otherwise
00090 %
