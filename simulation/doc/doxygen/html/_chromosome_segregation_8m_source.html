00005 % @description
00006 %   Biology
00007 %   ======================
00008 %   Chromosome segregation in Mycoplasma genitalium has not been well described.
00009 %   Chromosome segregation in M. genitalium is believed to result from two
00010 %   factors
00011 %   - entropically favorable segregation throughout DNA replication
00012 %   - four chromosome segregation proteins which act to decatenate the
00013 %     chromosomes following the completion of replication
00014 %   Little is known about the times at which these two factors contribute to
00015 %   chromosome segregation.
00016 %
00017 %   Bacterial chromosome segregation is believed to be entropically favorable
00018 %   and occur during replication (Bloom et al. 2010, Jun et al. 2006). That is,
00019 %   its believed to be entropically unfavorable for two chromosomes to be
00020 %   located near each other, and so as the replication fork moves, the
00021 %   replicated DNA molecules migrate away from each other toward the poles. We
00022 %   assume that by the end of replication the chromosomes have already migrated
00023 %   toward the poles.
00024 %
00025 %   Four proteins are believed to be involved in chromosome decatenation in M.
00026 %   genitalium:
00027 %   - MG_470_MONOMER  nucleotide binding domain protein CobQ/CobB/MinD/ParA
00028 %   - MG_221_OCTAMER  MraZ
00029 %   - MG_387_MONOMER  GTP binding protein Era
00030 %   - MG_384_MONOMER  GTPase Obg
00031 %   The specific function, kinetics, and metabolic costs of these proteins are
00032 %   not known. Furthemore, because M. genitalium contains a reduced complement
00033 %   of segregation proteins, it is difficult to infer their functions from
00034 %   studies of other bacterial species.
00035 %
00036 %   Knowledge Base
00037 %   ======================
00038 %   The knowledge base contains the value of the gtpCost parameter.
00039 %
00040 %   Representation
00041 %   ======================
00042 %   The substrates and enzymes properties represent the counts of available
00043 %   metabolites and segregation enzymes. gtpCost represents the energetic
00044 %   (GTP) cost of chromosome segregation. The segregation status of the
00045 %   chromosomes is represented as a boolean properties (segregated) of the
00046 %   Chromosome class.
00047 %
00048 %   Initialization
00049 %   ======================
00050 %   Chromosome initializes to a state with 1 chromosome (which obviously
00051 %   hasn't yet segregated from the second chromosome which will be later
00052 %   produced).
00053 %
00054 %   Simulation
00055 %   ======================
00056 %   Because little is known about the molecular biology of chromosome
00057 %   segregation, we have chosen to implement this process as a simple boolean
00058 %   rule: chromosome segregation can occur if:
00059 %   - the chromosome is replicated
00060 %   - the chromosome is properly supercoiled
00061 %   - there is at least one free molecule of each segregation proteins, and
00062 %   - there is at least gtpCost free GTP molecules
00063 %
00064 %   Note, although the Glass et al. gene essentiality study suggests that the
00065 %   cobQ/cobB/minD/parA gene is non-essential, we model this gene as essential
00066 %   because we don't know its specific function and how the other segregation
00067 %   proteins compensate in its absence.
00068 %

00070 %   ======================
00071 %   1. Bloom, K., Joglekar, A. (2010). Towards building a chromosome segregation
00072 %      machine. Nature 463: 446-456.
