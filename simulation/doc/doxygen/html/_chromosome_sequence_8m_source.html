00001 % ChromosomeSequence
00002 %   Object to represent the circular, complementary nature of chromosomal
00003 %   sequences. Externally this object appears as a circular len X 4 character
00004 %   array, where len is the length of the positive strand sequence, and the four
00005 %   columns represent: (1) mother chromosome, positive strand, (2) mother
00006 %   chromosome, negative strand, (3), daughter chromosome, positive strand,
00007 %   (4) daughter chromosome, negative strand. Subscript referencing is
00008 %   automatically wrapped over both dimensions so that
00009 %     (a) x(-1,1) references to x(end-1,1) and
00010 %     (b) x(1,5) references to x(1,1).
00011 %
00012 %   This object behaves differently that character arrays in several ways:
00013 %   - Subscript assignment is prohibited
00014 %   - Reshaping, squeezing, permuting, transposing, and concatenating return
00015 %     results NOT of type ChromosomeSequence, but of type char
00016 %
00017 %   Internally this class uses one property positiveStrandSequence to represent
00018 %   the mother and daughter chromosomes positive and negative sequences.
00019 %   Negative sequences are computed using the seqcomplement method of the
00020 %   bioinformatics toolbox.
