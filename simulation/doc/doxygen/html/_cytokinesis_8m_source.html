00003 % @wholeCellModelID Process_Cytokinesis
00004 % @name             Cytokinesis
00005 % @description
00006 %
00007 %   Cytokinesis is the division of the cytoplasm, achieved by the pinching of
00008 %   the cell membrane. At a high level, it occurs by the formation of a
00009 %   contractile Z ring along the interior surface of the cell membrane. The ring
00010 %   is comprised of GTP-activated ftsZ polymer filaments that bend when their
00011 %   GTP is hydrolized. Cytokinesis is thus a cycle of filament binding, bending,
00012 %   and dissociation.
00013 %
00014 %   This process is based on the model described in Li et al. 2007.
00015 %     1. Filaments bind the membrane end-to-end to form a regular polygon
00016 %        (inscribed in the pinched cell circumference). Two filaments bind at
00017 %        each polygon edge.
00018 %     2. If there was a previous cycle, then when at least one filament has
00019 %        bound each polygon edge, the last remaining ring of bent filaments from
00020 %        the previous cycle can begin to dissociate.
00021 %     3. When all polygon edges have two filaments bound and all residual bent
00022 %        filaments have dissociated, the bending of the edges of the newly
00023 %        completed polygon may begin. For simplicity, all of the GTP in the pair
00024 %        of filaments at a particular edge hydrolyze at the same time.
00025 %     4. When all the filaments have been bent, the bent filaments can begin
00026 %        dissociating -- but only one from each polygon edge. The other ring
00027 %        must remain to maintain the new smaller pinched circumference.
00028 %     5. When only one ring of bent filaments remains, the cycle repeats.
00029 %     6. Cytokinesis concludes when the pinched diameter is smaller than the
00030 %        length of one filament.
00031 %
00032 %   How bending works:
00033 %    - Filaments are bound in the straight configuration, forming a polygon
00034 %      inscribed in the cell circumference.
00035 %    - When the filaments bend, their length does not change. They bend just
00036 %      enough so that when they've all bent, a new circle is formed. Its
00037 %      circumference equals the old polygon's perimeter. Each fragment is now
00038 %      an arc.
00039 %
00040 %   In this version of the model, all filaments are of a fixed length. When they
00041 %   are joined end-to-end to form a regular polygon, the polygon does not quite
00042 %   fully inscribe the entire circumference. This remaining portion of the
00043 %   circumference does not bent when the polygon does. It is preserved and
00044 %   accounted for in the next iteration. For more details, see Li et al. 2007.
00045 %
00046 %   Note: Lluch-Senar et al have shown that M. genitalium cell division can occur 
00047 %   in the absence of FtsZ. Lluch-Senar showed that division occurs in ftsZ knockouts
00048 %   through motility.      
00049 %
00050 %   References
00051 %   ==========
00052 %   1. Lluch-Senar M, Querol E, Pinol J (2010). Cell division in a minimal bacterium 
00053 %      in the absence of ftsZ. Mol Microbiol. 78(2): 278-89. [PUB_0796]

00264         function result = calcNextPinchedDiameter(pinchedDiameter, filamentLengthInNm)
00265             import edu.stanford.covert.cell.sim.state.FtsZRing;
00266             
00267             numEdges = FtsZRing.calcNumEdges(pinchedDiameter, filamentLengthInNm);
00268             
00269             %the diameter of the circle that circumscribes the regular polygon
00270             %having numEdges sides
00271             flooredDiameter = filamentLengthInNm * 1e-9 / sin(pi/numEdges);
00272             
00273             %the diameter of the circle whose circumference consists of the
00274             %bound, bent filaments
00275             newDiameter = filamentLengthInNm * 1e-9 * numEdges / pi;
00276             
00277             %the parenthesized expression makes up for floor error in numEdges
00278             result = newDiameter + (pinchedDiameter - flooredDiameter);
00279             
00280             %if no additional pinching necessary set pinched diameter to zero
00281             if result <= filamentLengthInNm * 1e-9
00282                 result = 0;
00283             end
00284         end
