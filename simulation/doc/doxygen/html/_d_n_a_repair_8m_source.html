00024 %   Direct Damage Reversal (DDR)
00025 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

00027 %   ligase ligA (MG_254). ligA is capable of repairing single strand
00028 %   breaks by a NAD-dependent mechanism:
00029 %
00030 %         base    O   base    O           base    O   base    O
00031 %          |     ||    |     ||            |      ||   |     ||
00032 %   ...-O-sugar-O-P-O-sugar-O-P-O-  +  HO-sugar-O-P-O-sugar-O-P...  + NAD
00033 %                 |           |                   |           |
00034 %                 O           O                   O           O
00035 %
00036 %                                    ||

00038 %                                    V
00039 %

00042 %          |     ||    |     ||    |     ||    |    ||
00043 %   ...-O-sugar-O-P-O-sugar-O-P-O-sugar-O-P-O-sugar-O-P...  + AMP + NMN + H
00044 %                 |           |           |           |
00045 %                 O           O           O           O

00048 %   Base Excision Repair (BER)
00049 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00050 %   Repairs specific spontaneously and endogenously produced DNA damaged
00051 %   including base modifications, abasic sites, and single strand breaks
00052 %   by the following mechanism:
00053 %   1. Base excision by Fpg (MG_498) or Ung (MG_097) glycosylase
00054 %
00055 %         base    O                 OH      O
00056 %          |     ||                  |     ||
00057 %      -O-sugar-O-P... + H2O ==> -O-sugar-O-P  +  base
00058 %                 |                         |
00059 %                 O                         O
00060 %
00061 %   2. a. i.  3'-AP lyase Fpg (MG_498)
00062 %         ii. 5'-AP endonuclease Nfo (MG_235)
00063 %      b. i.  5'-deoxyribosephosphodiesterase Nfo (MG_235)
00064 %         ii. 3'-(deoxyribose-5'-phosphate) lyase dnaN (MG_001)
00065 %   3. 5'-3' Polymerization by dnaN (MG_001)
00066 %   4. Ligation by ligA (MG_254). See DDR section for details.
00067 %   5. Catabolism and export of damaged nucleobases and nucleosides by
00068 %      metabolic process.
00069 %
00070 %   Flap endonuclease fen1 homologs have not been identified in M.
00071 %   genitalium; MG_262 is a weak fen1 homolog, but is more strongly
00072 %   homologous to polI exonucleases. Consequently, we do not model flap
00073 %   base excision repair.
00074 %
00075 %   Global Genomic (GGR) Nucleotide Excision Repair (NER)
00076 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00077 %   NER recognizes bulky distortions in the shape of the DNA helix caused
00078 %   by damaged bases and cross links caused by NO and UV light, and uses
00079 %   endonucleases with lower specificity than that of BER. Consequently,
00080 %   NER has broader repair capability than base excision repair. Here we
00081 %   model only global genomic NER. In some bacteria such as E. coli DNA
00082 %   damage is sensed by RNA polymerase and coupled to the NER machinery by
00083 %   the transcription-repair coupling factor mfd.
00084 %
00085 %   1. UvrABC (MG_073, MG_206, MG_421) cleaves DNA 3' and 5' to lesion
00086 %   2. Helicase PcrA (MG_244) excises cleaved DNA
00087 %   3. Cleavage excised oligonucleotide (12-13 nts) by MgpA (MG_190)
00088 %   4. 5'-3' Polymerization by dnaN (MG_001)
00089 %   5. Ligation by ligA (MG_254). See DDR section for details.
00090 %   6. Catabolism and export of damaged nucleobases and nucleosides by
00091 %      metabolic process.
00092 %
00093 %   Note steps (3) and (4-5) occur in parallel. NER removes 2.7 x 10^3
00094 %   excison products per cell per minute.
00095 %
00096 %   Homologous Recombination (HR) Double Strand Break Repair (DSBR)
00097 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00098 %   HR repairs double strand breaks caused by ionizing radiation and
00099 %   stalled replicaton forks. HR also repairs strand gaps which can be
00100 %   generated by the interaction of replication forks with unrepaired
00101 %   lesions. M. genitalium has a very reduced complement of recombination
00102 %   repair proteins. Normally recombination repair involves the following
00103 %   steps:
00104 %   1. Initiation: 5'-3' exonuclease removes dNMPs from 5' ends of strands
00105 %      leaving 3' overhangs. Daley and Wilson have shown homologous
00106 %      recombination is most efficient with overhangs of at least 8 bases
00107 %      [6]. No tradiational initiation gene has been identified in M.
00108 %      genitalium. 5'-3' resection could be carried out by pcrA (MG_244)
00109 %      or polI-like (MG_262). Here we model pcrA as the NER helicase, and
00110 %      therefore choose to model polI-like as the HR initiator.
00111 %   2. Strand exchange: One of the damaged 3' overhang invades the
00112 %      undamaged homologous chromosome, forming a holliday junction.
00113 %      Catalyzed by recA (MG_339)
00114 %   3. 5'-3' Polymerization by dnaN (MG_001)
00115 %   4. 2 ligations by ligA (MG_254). See DDR section for details.
00116 %   5. Second strand exchange: The second damaged 3' overhang invades the
00117 %      undamaged homologous chromosome, forming a second holliday
00118 %      junction. Also catalyzed by recA (MG_339)
00119 %   6. Holliday junction migration: ruvAB (MG_358, MG_359) widens distance
00120 %      between the two strand cross over points by moving holliday
00121 %      junctions to preferred sequence 5'-(A/T)TT?(G/C)-3' [PUB_0532].
00122 %   7. Resolution: recU (MG_352) dimer nicks DNA at cross over points
00123 %      creating four single strand breaks
00124 %   8. 4 ligations by ligA (MG_254). See DDR section for details.
00125 %   9. Catabolism and export of damaged nucleobases and nucleosides by
00126 %      metabolic process.
00127 %
00128 %   Additionally, homologous recombination is believed to be critical
00129 %   for antigenic variation in M. genitalium (Sluijter et al).
00130 %
00131 %   DNA integrity scanning protein DisA
00132 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00133 %   In B. subtilis DisA recognizes strand breaks, base damage,
00134 %   and cross links (possibly through ssDNA, Holliday junctions, or stalled
00135 %   replication forks that are generated during recombination repair of
00136 %   these lesions), binds damaged DNA, and via an unknown mechanism delays
00137 %   sporulation. DisA has weak, non-specific DNA binding. In addition DisA
00138 %   has diadenylate cyclization (produces c-di-AMP from 2 ATPs) activity
00139 %   which is abolished when bound to branched DNA such as Holliday
00140 %   junctions. c-di-AMP has been speculated as the second messenger which
00141 %   couples DisA damage recognition to regulation of the transcriptional
00142 %   regulator Spo0A, and delayed sporulation. The rest of the B. subtilis
00143 %   signaling system (spo0A, kinA, kinE) is not present in M. genitalium.
00144 %
00145 %   DNA Restriction/Modification
00146 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00147 %   Organisms employ a restriction/modification systems to distinguish
00148 %   self from foreign DNA. These systems methylate hemi-methylated DNA and
00149 %   cleave unmethylated DNA. The systems maintains self DNA in the fully
00150 %   methylated state so that it can be distinguished from foreign
00151 %   DNA (which will be unmethylated) on methylation status.
00152 %
00153 %   M. genitalium contains a reduced restriction/modification system
00154 %   consisting of the EcoD type I DNA recognition subunit (MG_438) and the
00155 %   MunI type II methylase (MG_184). M. genitalium does not contain a
00156 %   type III R/M system.
00157 %
00158 %   EcoD is the DNA binding domain of a multimetric methylation and
00159 %   restriction complex. Type I R/M systems typically have structure R2M2S
00160 %   or M2S. EcoD recognizes the sequence 5'-TCARTTC-3'. The other subunits
00161 %   methylate the N(7) position of A-3, producing N(7)-methyladenine or
00162 %   cleavage DNA at this position. M. genitalium contains only the DNA
00163 %   recognition subunit. 
00164 %
00165 %   MunI methylase (MG_184) methylates the N(6) position of A-3 in the
00166 %   sequence 5'-CAATTG-3', producing N(6)-methyladenine. M. genitalium
00167 %   does not contain a MunI endonuclease. Type II R/M systems have been
00168 %   found as monomers, dimers, and trimers.
00169 %
00170 %   DNA repair mechanisms not present in M. genitalium
00171 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00172 %   - Transcription-coupled NER (TC-NER)
00173 %     M. genitalium does not have a transcription-repair coupling factor
00174 %     mfd homolog.
00175 %   - Additional eukaryotic homologous recombination (HR) pathways
00176 %     - Synthesis-dependent strand annealing (SDSA)
00177 %     - Single-strand annealing (SSA)
00178 %     - Break-induced replication (BIR)
00179 %   - Non-homologous end joining (NHEJ)
00180 %     M. genitalium does not have a dedicated NHEJ DNA ligase ligD or a
00181 %     DNA-end-binding protein Ku.
00182 %   - Mismatch Repair
00183 %   - SOS Response
00184 %
00185 %   Metabolism of damaged nucleo-bases, -sides, and -tides
00186 %   +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
00187 %   No gene has been identified which cleaves DNA oligonucleotides;
00188 %   possible genes include pcrA (MG_244), mgpA (MG_190), and polI-like
00189 %   (MG_262). Here we model pcrA as the NER helicase and polI-like as the
00190 %   HR resection enzyme, leaving mgpA without an assigned function.
00191 %   Therefore we choose to model mgpA as capable of cleaving
00192 %   oligonucleotides released by NER into single nucleotides.
00193 %
00194 %   Nucleotides released by polI-like or mgpA are returned to the
00195 %   cytoplasmic free pool, and are available to the metabolism process.
00196 %   Damaged nucleotides are degraded to nucleosides and are exported by
00197 %   the metabolism process. Damaged nucleobases are also exported by the
00198 %   metabolism process.

00219 %   =====================================================================
00220 %   8 properties are used to keep track of the damage state of DNA. These
00221 %   properties are owned by the DNA damage process, and are copied locally at
00222 %   each time step. The first  dimension of each has length equal to that of
00223 %   the chromosome.
00224 %   - occupiedDNA               enumeration   proteins bound to DNA
00225 %   - gapSites                  true/false    sugar gaps
00226 %   - abasicSites               true/false    abasic sites
00227 %   - damagedSugarPhosphates    enumeration   damaged sugar phosphates
00228 %   - damagedBases              enumeration   damaged bases
00229 %   - intrastrandCrossLinks     true/false    intrastand cross link
00230 %   - strandBreaks              true/false    strand break (missing sugar-phosphate bond)
00231 %   - hollidayJunctions         true/false    holliday junction
00232 %
00233 %   substrates represents the amounts of free metabolites. initializeConstants
00234 %   synchronizes the substrates of this process with that of the DNA damage
00235 %   process to make it easy for these processes to communicate. That is
00236 %   initializeConstants sets the substrates of both processes equal to the union
00237 %   of the two, are orders the substrates the same in both processes. Thus
00238 %   substrate indices defined in process are valid on properties received from
00239 %   the other process, and it is unecessary to remap properties in
00240 %   copyFromState.
00241 %
00242 %   enzymes represents the amount of DNA repair enzymes.
00243 %
00244 %   Rather than using reactionStoichiometryMatrix directly this process
00245 %   factors the reactionStoichiometryMatrix into two parts:
00246 %   - reactionSmallMoleculeStoichiometryMatrix: the stochiometry of free
00247 %     metabolites in repair reactions
00248 %   - reactionDNAStoichiometryMatrix: the stoichiometry of DNA bases in
00249 %     repair reactions
00250 %
00251 %   In addition, rather than using reactionModificationReaction to
00252 %   represent the macromolecule affected by each reaction we use
00253 %   reactionDNABase to represent the DNA base affected by each repair reaction.
00254 %
00255 %   reactionCatalysisMatrix represents the enzymes which catalyze each repair
00256 %   reaction. enzymeBounds represents the backward and forward enzyme rate
00257 %   constant of each repair reaction.

00260 %   about the specific action of each enzyme.

00262 %   Simulation
00263 %   =====================================================================
00264 %   The simulation is implemented as six subfunctions which are called in a
00265 %   random order from evolveState:
00266 %   - base excision repair
00267 %   - nucleotide excision repair
00268 %   - homologous recombination double strand break repair
00269 %   - polymerization/ligation
00270 %   - DisA damage recognition, and
00271 %   - restriction/modification
00272 %
00273 %   Each subfunction carries out some variation of the algorithm:
00274 %   1. Identify chromosomes sites that are substrates for the repair system
00275 %   2. Eliminate sites that neighbor damage, or which are occupied by proteins
00276 %      (other than the DNA damage scanning protein DisA).
00277 %   3. Compute maximum number of repair reactions that can occur based on
00278 %      metabolite and enzyme availability and kinetics.
00279 %   4. Randomly select among chromosome sites
00280 %   5. Update state of DNA. Update metabolites.
00281 %   6. Free any DisA that was previously bound to the repaired site.

01098             c = this.chromosome;
01099             
01100             %% stop early
01101             %no enzyme
01102             rxIdxs = [this.reactionIndexs_NER_incision; this.reactionIndexs_NER_excision; this.reactionIndexs_DNACleavage];

01105             end
01106             
01107             %no damage
01108             if nnz(c.damagedBases) == 0 && nnz(c.intrastrandCrossLinks) == 0
01109                 return;
01110             end
01111 
01112             %% damaged sites: positions and strands

01114                 find(xor(c.damagedBases, c.intrastrandCrossLinks)), ...
01115                 find(c.damagedBases == this.substrateGlobalIndexs(this.substrateIndexs_m6AD)), ...
01116                 'rows');
01117             if isempty(damagedPositionsStrands)
01118                 return;
01119             end
01120 
01121             %% exclude methylated restriction/modification sites

01123             % methylation of hemimethylated restriction/modification sites is
01124             % greater their rate of demethylation by the DNA repair machinery

01126                 reshape(this.RM_MunI_RecognitionSites(:, this.RM_MunI_MethylatedPositions), [], 1) ...

01143             
01144             idxs = find(c.intrastrandCrossLinks(damagedPositionsStrands));

01172

01174             rxCounts(1) = numReactions;

01177                 reshape(repmat(positionsStrands(:, 1), 1, numExcisions) + repmat(excisedPositions - 1, numReactions, 1), [], 1), ...

01214             c.strandBreaks(          incisedPositionsStrands) = 1;
01215             c.gapSites(              excisedPositionsStrands) = 1;
01216             c.abasicSites(           excisedPositionsStrands) = 1;
