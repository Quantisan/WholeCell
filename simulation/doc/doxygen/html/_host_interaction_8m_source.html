00006 %   Introduction
00007 %   ======================
00008 %   Medical relevance
00009 %   ++++++++++++++++++++++
00010 %   14 Mycoplasmas and 2 ureaplasmas have been detected in humans
00011 %   [PUB_0840]. 6 of these species, including M. genitalium have been found
00012 %   in the urogenital tract. M. genitalium is a common urogenital pathogen,
00013 %   and one of the most common causes of non-gnoccocal urethritis. M.
00014 %   genitalium is estimated to colonize the urogenital tract of 3% of all
00015 %   women [PUB_0840], and up to 9-10% in some populations [PUB_0841]. Most
00016 %   M. genitalium infections are asymptomatic, and many researchers have
00017 %   written that M. genitalium is close to "ideal parasite" [PUB_0313],
00018 %   typically living in  harmony with its host. A small fraction of M.
00019 %   genitalium infections manifest as urethritis, cervicitis, and pelvic
00020 %   inflammatory disease.
00021 %
00022 %   Pathophysiology
00023 %   ++++++++++++++++++++++
00024 %   Little is known about the pathophysiology of M. genitalium. The
00025 %   literature on M. genitalium pathophysiology has focused on a few
00026 %   topics:
00027 %   1. Adhesion
00028 %      Several studies have suggested that the terminal organelle
00029 %      lipoproteins MG_191, MG_192, MG_217, MG_318, and MG_386 are involved
00030 %      in adhesion [PUB_0088, PUB_0313]
00031 %   2. Host immune response activation
00032 %      Several studies have suggested that the gene products of MG_149,
00033 %      MG_309, and MG_412 activate the host NF-kB response through
00034 %      interaction with host TLR receptors 1, 2, and 6 [PUB_0194, PUB_0842,
00035 %      PUB_0844].
00036 %   3. Antigenicity and antigenic variation
00037 %      Several studies have focused on antigenticity and antigenic
00038 %      variation. Razin reported that MgPa (MG_191) is one of the main
00039 %      antigenic components, and that antibodies against it may be found in
00040 %      infected patients [PUB_0088]. Shimizu et al have observed two
00041 %      differentially expressed forms of MG_149 -- P18 and P20 [PUB_0844].
00042 %   4. Toxic byproducts
00043 %      M. genitalium byproducts including hydrogen peroxide and superoxide
00044 %      radicals have been suggested to cause oxidative damage to host
00045 %      membranes [PUB_0313].
00046 %
00047 %   The following table summarizes genes that have been implicated
00048 %   interactions with mammalian hosts.
00049 %
00050 %     Gene    Symbols                  Localization         Function
00051 %     ======  =======================  ============         ===========
00052 %     MG_075                           lipoprotein          antigen
00053 %     MG_149  N-ALP2, P18, P20         lipoprotein          activates NF-kB as TLR 1/2 ligand
00054 %     MG_191  MgPa, P140               Trm Org lipoprotein  adhesion
00055 %     MG_192  P110, P40, P90, pre-B/C  Trm Org lipoprotein  adhesion
00056 %     MG_200                           cytosol              motility
00057 %     MG_217  P65                      Trm Org lipoprotein  adhesion
00058 %     MG_218  HMW2                     Trm Org core         motility, attachment
00059 %     MG_288                           cytosol              homologous to protein L, but annotation as protein L is inconsistent with cytosolic localization; not currently modeled as participating in interaction with host  
00060 %     MG_309                           lipoprotein          activates NF-kB as TLR 2/6 ligand
00061 %     MG_312  HMW1, P65                Trm Org core         motility, attachment
00062 %     MG_317  HMW3, P200, P69          Trm Org core         motility, attachment
00063 %     MG_318  P30, P32                 Trm Org lipoprotein  adhesion
00064 %     MG_386  P200                     Trm Org lipoprotein  adhesion
00065 %     MG_412  N-ALP1                   lipoprotein          activates NF-kB as TLR 1/2 ligand
00066 %
00067 %   Diagnosis
00068 %   ++++++++++++++++++++++
00069 %   M. genitalium is typically not directly diagnosed, but rather is
00070 %   diagnosed by negative results for other urogenital pathogen. Nucleic
00071 %   acid amplification testing (NAAT) is commonly used to rule out
00072 %   chlamydia infection, and suggest M. genitalium infection. Less commonly
00073 %   NAAT is used directly to diagnose M. genitalium infection.
00074 %
00075 %   Treatment
00076 %   ++++++++++++++++++++++
00077 %   M. genitalium is most commonly treated using macrolides such as
00078 %   azithromycin and tetracycline [PUB_0840]. Azithromycin-resistant M.
00079 %   genitalium may be treated with fluoroquinolones [PUB_0840].
00080 %   Taylor-Robinson and Bebear have catalogued the susceptibility of M.
00081 %   genitalium to 18 antibiotics [PUB_0846].
00082 %
00083 %   Tetracycline and aminoglycoside antibiotics inhibit the 30S ribosomal
00084 %   particle. Macrolides and lincosamide antibiotics inhibit the 50S
00085 %   ribosomal particle. Fluoroquinolone antibiotics inhibit topoisomerase
00086 %   II/DNA gyrase and topoisomerase IV.
00087 %
00088 %   Knowledge Base
00089 %   ======================
00090 %   The antibiotic susceptibilities of M. genitalium proteins are encoded
00091 %   as protein activation rules in the knowledge base, and are simulated by
00092 %   the ProteinActivation class. 32 antibiotics are represented as
00093 %   metabolites in the knowledge base.
00094 %
00095 %   Representation
00096 %   ======================
00097 %   The state of the M. genitalium proteins involved in interaction with
00098 %   the mammalian urogenital epithelium is represented by the enzymes
00099 %   property. This property contains the current copy number of each
00100 %   protein.
00101 %
00102 %   The state of the host mammalian urogenital epithelium is represented by
00103 %   4 boolean properties in the host class:
00104 %   - isBacteriumAdherent: indicates whether or not cell is adherent to the epithelial surface
00105 %   - isTLRActivated: indicates the state of host TLRs 1, 2, and 6
00106 %   - isNFkBActivated: indicates the state of host NF-kB
00107 %   - isInflammatoryResponseActivated: indicates the state of the host inflammatory response
00108 %
00109 %   Initialization
00110 %   ======================
00111 %   The state of the M. genitalium host interaction proteins is initialized
00112 %   by the initializeState method of the Simulation class. All proteins are
00113 %   initialized to the mature state and localization.
00114 %
00115 %   The state of the host urogenital epithelium is initialized using the
00116 %   same Boolean rules as in the evolveState method. That is we assume that
00117 %   the urogenital epithelium is in a steady-state with respect to its
00118 %   interaction with M. genitalium.
00119 %
00120 %   Simulation
00121 %   ======================
00122 %   Very little quantitative is available for the interaction between M.
00123 %   genitalium and its mammalian host. The little data available is
00124 %   patchwork and qualitative. Thus the model here for the interaction of
00125 %   M. genitalium with its host is necessarily qualitative.
00126 %
00127 %   The model consists of several Boolean rules which calculate the value
00128 %   of the 4 properties of the Host class:
00129 %   1. M. genitalium is adherent if the terminal organelle is properly
00130 %      formed and all the adhesion proteins are expressed.
00131 %   2. TLRs 1, 2, and 6 are activated if M. genitalium is adherent (see
00132 %      rule 1) and the M. genitalium lipoprotein ligands are expressed.
00133 %   3. NF-kB is activated if TLRs 1 and 2 or TLRs 2 and 6 are activated.
00134 %   4. The host inflammatory response is activated if NF-kB is activated or
00135 %      if MG_075 is present.
00136 %
00137 %   The model assumes that the mammalian urogenital epithelium reaches a
00138 %   steady-state quickly with respect to its interaction with M.
00139 %   genitalium.
