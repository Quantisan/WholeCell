00003 % @wholeCellModelID Process_ProteinActivation
00004 % @name             Protein Activation
00005 % @description
00006 %   Biology
00007 %   ================================
00008 %   The activity of proteins and other macromolecules can be modulated by other
00009 %   molecules such as free metabolites (and pseudo metabolites we've represented
00010 %   as stimuli) both at the enzymatically active site, and at more distant
00011 %   allosteric sites. This regulation helps proteins, and thereby the cell,
00012 %   respond to changes in the internal and external environments, and maintain
00013 %   homeostasis. From a network perspective, this regulation can give rise
00014 %   to positive and negative feedback loops.
00015 %
00016 %   This process models protein regulation by transitioning proteins between
00017 %   enzymatically active (simulation properties matureMonomers and
00018 %   matureComplexs) and inactive states (simulation properties inactiveMonomers
00019 %   and inactiveComplexs) according to boolean regulatory rules built from the
00020 %   primary literature. Regulatory rules are evaluated independently for each
00021 %   compartment. Proteins for which we have not implemented a regulatory
00022 %   rule are assumed to always remain in the enyzmatically active state.
00023 %
00024 %   Knowledge Base
00025 %   ================================
00026 %   As of 8/9/2010 the M. genitalium knowledge base includes regulatory rules
00027 %   for six proteins:
00028 %   - MG_085_HEXAMER  HPr(Ser) kinase/phosphatase
00029 %   - MG_101_MONOMER  Uncharacterized HTH-type transcriptional regulator
00030 %   - MG_127_MONOMER  Spx subfamily protein
00031 %   - MG_205_DIMER    heat-inducible transcription repressor HrcA, putative
00032 %   - MG_236_MONOMER  ferric uptake repressor
00033 %   - MG_409_DIMER    phosphate transport system regulatory protein PhoU,
00034 %                     putative
00035 %
00036 %   Boolean activation rule syntax
00037 %   ++++++++++++++++++++++++++++++++
00038 %   - Boolean activation rules must evaluate to true or false where true
00039 %     indicates that the protein species is active, and false indicates
00040 %     the protein species is inactive. That is protein species are
00041 %     activated/inactivated in an all-or-nothing fashion for each
00042 %     compartment.
00043 %   - Using Whole Cell Model IDs boolean rules can contain references to
00044 %     counts/concentrations of several kinds of objects:
00045 %     - stimuli                    count                (simulation.stimuli)
00046 %     - metabolites                concentration (mM)   (simulation.metabolites)
00047 %     - mature protein monomers    concentration (mM)   (simulation.matureIndexs)
00048 %     - mature protein complexes   concentration (mM)   (simulation.matureComplexIndexs)
00049 %     That is, Whole Cell Model IDs of knowledge base objects contained
00050 %     within boolean rules will replaced by the current value or
00051 %     concentration in mM of the corresponding knowledge base object before
00052 %     evaluation of the boolean rule
00053 %   - Boolean activation rules permit the operators: |, &, +, -, !, <, >, <=, >=, ==
00054 %   - Boolean activation rules permit parenthesis for grouping
00055 %   - Boolean activation rules permit spaces
00056 %
00057 %   Examples
00058 %   - objectID1 > val1
00059 %   - objectID1 <= val1
00060 %   - objectID1 == val1 | objectID2>conc2
00061 %   - objectID1 == val1 | !(objectID2>conc2 & objectID3<=conc3 & (objectID4+object5)<conc5)
00062 %
00063 %   Representation
00064 %   ================================
00065 %   The substrates and inactivatedSubstrates properties represent the counts of
00066 %   enzymatically active proteins in each compartment (all of the compartments
00067 %   in the simulation are separately mapped to this process). stimuli represents
00068 %   the  values of external perturbations and pseudo free metabolites
00069 %   (simulation property stimuli), free metabolites, and macromolecules which
00070 %   affect the activity of the proteins.
00071 %
00072 %   activationRules represents the boolean regulatory rules governing each of
00073 %   the proteins in substrates. The activation rule for each protein is
00074 %   evaluated independently for each compartment. Boolean rules are transcoded
00075 %   to MATLAB syntax from the syntax described above during initializeConstants.
00076 %   Boolean rules are evaluated using the MATLAB eval command in a workspace
00077 %   where variables having the names of whole cell model ids of the stimuli are
00078 %   defined and set to the value/concentration of the corresponding stimulus.
00079 %
00080 %   Initialization
00081 %   ================================
00082 %   The same boolean regulatory rules that are evaluated during the simulation
00083 %   are evaluated during initialization. Proteins for which their regulatory
00084 %   rule evaluates to false are initialized to the inactive state
00085 %   (inactivatedSubstrates); proteins for which their regulatory rule evaluates to
00086 %   true are initialized to the active state (substrates). This is achieved by
00087 %   calling the evolveState method.
00088 %
00089 %   Simulation
00090 %   ================================
00091 %   For each compartment
00092 %     1. Use scaleComponents to compute the concentrations of objects (except
00093 %        stimuli objects) mapped to the process's stimuli property.
00094 %     2. Assign to local variables named with the stimuli Whole Cell Model IDs
00095 %        values equal that of the corresponding object (for stimuli objects) or
00096 %        the concentration (for all other objects) of the corresponding object
00097 %        computed in (1).
00098 %     3. Evaluate boolean regulatory rule of each protein using the MATLAB eval
00099 %        command.
00100 %     4. Update substrates and inactivatedSubstrates properties based on (3).
00101 %        Proteins whose boolean regulation rule evaluates to false are
00102 %        transitioned from substrates to inactivatedSubstrates. Proteins whose
00103 %        boolean regulation rule evaluates to true undergo the opposite
00104 %        transition.
00105 %   End
