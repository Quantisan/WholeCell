00001 %Protein Decay
00002 %
00003 % @wholeCellModelID Process_ProteinDecay
00004 % @name             Protein Decay
00005 % @description
00006 %   Biology
00007 %   ===========
00008 %   Poisson simulation of protein damage, repair, and decay mediated by
00009 %   - chaperone: clpB chaperone refolds/disaggregates proteins by an
00010 %     energy-dependent mechanism
00011 %   - proteases: lon, ftsH; each is believed to processively cleave
00012 %     peptides into smaller peptides of approximately 20 amino acids
00013 %     - ftsH: cleaves peptides with tmRNA stalled translation proteolysis
00014 %       tag; parameterized by
00015 %       - fragment length (aa)
00016 %       - kinetic rate (cleaves/s)
00017 %       - energy per cleavage (ATP/cleavage)
00018 %     - lon: cleaves all other peptides; parameterized by
00019 %       - fragment length (aa)
00020 %       - kinetic rate (cleaves/s)
00021 %       - energy per cleavage (ATP/cleavage)
00022 %   - several peptidases: the specific functions and kinetics of the
00023 %     individual peptidases are unknown and not modelled; decay proceeds as
00024 %     long as at least one of each peptidase is present
00025 %
00026 % 1. Misfold proteins
00027 %    - Protein misfolding is modeled as a poisson process
00028 %      with a small rate constant
00029 % 2. Refold cytosolic proteins
00030 %    - Occurs if clpB is present
00031 % 3. Decay macromolecular complexes
00032 %    - Model decay is poisson process with rate parameter given by the
00033 %      inverse weighted average half life of the complex's subunits
00034 %    - Salvage bound prosthetic groups
00035 %    - Mark subunits as damaged to be degraded by either the
00036 %      protease/peptidase or ribonuclease machinery
00037 % 4. Decay protein monomers and proteolysis tagged polypeptides
00038 %    - Model selection of monomers that decay as poisson process with rate
00039 %      parameters equal to the decay rates (ln(2)/half life as computed by
00040 %      the N-end rule) of the monomers.
00041 %    - Execute decay reactions as long as the following are available
00042 %      - Lon/FtsH protease for normal / ribosome-stalled peptides
00043 %      - Complete peptidase complement
00044 %      - Water for hydrolysis of peptide bonds
00045 %
00046 %   Knowledge Base
00047 %   ===============
00048 %   The knowledge base contains implements the N-end rule which predicts the
00049 %   half life of every protein monomer based on the average experimentally
00050 %   measured (in E. coli) protein half lives for each possible N-terminal amino
00051 %   acid. Exceptions:
00052 %   - signal sequences are assumed to have 0 half life
00053 %   - proteolysis tagged polypeptides are assumed to have 0 half life
00054 %
00055 %   The knowledge base predicts the half lives of complexes as the weighted mean
00056 %   of that of their constituent protein monomers and RNAs.
00057 %
00058 %   The knowledge base ProteinMonomer and ProteinComplex classes also computes
00059 %   hydrolytic degradation reactions of every protein monomer and complex:
00060 %   - the (modified) amino acids released by hydrolytic cleavage of every
00061 %     protein monomer
00062 %   - water required for hydrolytic peptide bond cleavage
00063 %   - prosthetic groups released by cleavage
00064 %
00065 %   The ProteinMonomer class is also used to compute hydrolytic degradation
00066 %   reactions for proteolysis tagged polypeptides.
00067 %
00068 %   Representation
00069 %   ===============
00070 %   The substrates, enzymes, monomers, RNAs, and complexs properties represent
00071 %   the counts of metabolites, proteases and peptidases, protein monomers,
00072 %   damaged RNAs, protein complexes. The substrates and enzymes properties have
00073 %   only 1 compartment (cytosol). The monomers, RNAs, and complexs properties
00074 %   have 5 compartments. The monomers and complexs properties represent all
00075 %   forms of proteins (nascent, mature, damaged, folded, misfolded, etc.)
00076 %
00077 %   abortedSequences represents the sequences of every proteolysis tagged
00078 %   monomer in the cytosol.
00079 %
00080 %   The process contains intermediate representation of protein complex
00081 %   degradation, but not of protein monomer degradation, misfolding, or
00082 %   refolding. Degrading complexes are broken up into the constituent RNAs and
00083 %   monomers, and these components are marked as "damaged" and recognized by RNA
00084 %   and protein monomer degradation as molecules with 0 half life. Protein
00085 %   monomer degradation is treated as an all-or-nothing event that either
00086 %   proceeds to complete with a time step or doesn't progress at all.
00087 %
00088 %   proteinMisfoldingRate represents the rate at which every protein misfolds in
00089 %   seconds.
00090 %
00091 %   monomerDecayRates and complexDecayRates represent the decay rate of each
00092 %   protein monomer and complex species in seconds. misfolded and damaged
00093 %   proteins, signal sequences, and proteolysis tagged polypeptides have 0 half
00094 %   lives. All other proteins have positive half lives.
00095 %
00096 %   monomerDecayReactions and complexDecayReactions represent the metabolites
00097 %   required and released by hydrolytic cleavage of protein monomers and the
00098 %   break down of macromolecular complexes into their constituent subunits and
00099 %   sequestered prosthetic groups. monomerDecayReactions and
00100 %   complexDecayReactions are computed by the knowledge base ProteinMonomer and
00101 %   ProteinComplex classes.
00102 %
00103 %   Initialization
00104 %   ===============
00105 %   All protein monomers and complexs are initialized to the mature state. This
00106 %   is accomplished by the simulation class initializeState method.
00107 %
00108 %   Simulation
00109 %   ===============
00110 %   This process models misfolding, refolding, and protein monomer and complex
00111 %   degradation as an enyzme-dependent (excepct misfolding) poisson processes
00112 %   with rate parameter:
00113 %      lambda = proteins .* decayRates * stepSizeSec
00114 %
00115 %   Algorithm
00116 %   +++++++++++++++
00117 %   Each of misfolding, refolding, protein complex degradation, proteolysis
00118 %   tagged monomer degradation, and protein monomer degradation use a variant of
00119 %   the algorithm:
00120 %
00121 %   1. Stochastically select proteins to misfold/refolding/degrade based on
00122 %      poission distribution with
00123 %         lambda = proteins .* rates * stepSizeSec
00124 %   2. Limit refolding/degradation by availability of water and energy
00125 %   3. Limit protein refolding/degradation by available enzyme activity
00126 %      a. Refolding requires ClpB
00127 %      b. Protein monomer degradation requires Lon protease and 6 peptidases
00128 %      c. Proteolysis tagged polypeptide degradation requires FtsH protease and
00129 %         6 peptidases
00130 %   4. Update counts of proteins
00131 %   5. Update counts of metabolic reactants and byproducts of protein
00132 %      refolding/degradation
00133 %
00134 %   Compartments
00135 %   +++++++++++++++
00136 %   - Misfolding: occurs in all compartments
00137 %   - Refolding: occurs only in compartments where ClpB is present, that is the
00138 %     cytosol (and terminal organelle cytosol)
00139 %   - Complex degradation: occurs only in compartments where the protein decay
00140 %     machinery (Lon protease and peptidases) are presents, that is the cytosol
00141 %     (and terminal organelle cytosol)
00142 %   - Proteolysis tagged polypeptide degradation: only exist in cytosol, and are
00143 %     only degraded their (assumes they are accessible from within the cytosol
00144 %     to the integral membrane FtsH protease)
00145 %   - Protein monomer degradation:  occurs only in compartments where the Lon
00146 %     protease and peptidases are present, that is the cytosol.
00147 %
00148 % Author: Jonathan Karr, jkarr@stanford.edu
00149 % Affilitation: Covert Lab, Department of Bioengineering, Stanford University
00150 % Last updated: 8/4/2010

00590         %Model protein refolding as a binary metabolite-limited process:
00591         %If active ClpB is present in the cytosol and ATP is present, refold
00592         %all misfolded cytoplasmic proteins up to the amount of available ATP.
00593         %
00594         %Because the ATP cost of ClpB-mediated protein refolding/disagggregation
00595         %is unknown we assume that 1 ATP is required per refolding.
00596         %
00597         %Refolding proteins transition to the active state.
00598         function evolveState_RefoldProteins(this)
00599             %cumulative sum of misfolded monomers and complexes
00600             if ...
00601                     ~any(this.monomers(this.monomer.misfoldedIndexs, this.compartment.cytosolIndexs))                  && ...
00602                     ~any(this.monomers(this.monomer.misfoldedIndexs, this.compartment.terminalOrganelleCytosolIndexs)) && ...
00603                     ~any(this.complexs(this.complex.misfoldedIndexs, this.compartment.cytosolIndexs))                  && ...
00604                     ~any(this.complexs(this.complex.misfoldedIndexs, this.compartment.terminalOrganelleCytosolIndexs))
00605                 return;
00606             end
00607             
00608             %indices
00609             cIdxs = [this.compartment.cytosolIndexs; this.compartment.terminalOrganelleCytosolIndexs];
00610             reactantIdxs = [this.substrateIndexs_atp; this.substrateIndexs_water];
00611             productIdxs  = [this.substrateIndexs_adp; this.substrateIndexs_phosphate; this.substrateIndexs_hydrogen];
00612             
00613             %stochastically select proteins to refold, up to limit of available
00614             %reactants
00615             cumsumMisfoldedProteins = cumsum([0;
00616                 this.monomers(this.monomer.misfoldedIndexs, this.compartment.cytosolIndexs);
00617                 this.monomers(this.monomer.misfoldedIndexs, this.compartment.terminalOrganelleCytosolIndexs);

00621                 this.substrates(reactantIdxs)
00622                 cumsumMisfoldedProteins(end)]);
00623             if numFoldingProteins == 0
00624                 return;
00625             end
00626             
00627             foldingProteinIdxs = this.randStream.randperm(cumsumMisfoldedProteins(end));
00628             
00629             foldingMonomers = zeros(size(this.monomer.matureIndexs,1), 2);
00630             foldingComplexs = zeros(size(this.complex.matureIndexs,1), 2);
00631             
00632             for i = 1:numFoldingProteins

00634                 if j <= numel(this.monomer.matureIndexs)
00635                     foldingMonomers(j,1) = foldingMonomers(j,1)+1;
00636                 elseif j <= 2*numel(this.monomer.matureIndexs)
00637                     foldingMonomers(j - numel(this.monomer.matureIndexs), 2) = foldingMonomers(j - numel(this.monomer.matureIndexs), 2) + 1;

00639                     foldingComplexs(j - 2*numel(this.monomer.matureIndexs), 1) = foldingComplexs(j - 2*numel(this.monomer.matureIndexs), 1) + 1;
00640                 else
00641                     foldingComplexs(j - 2*numel(this.monomer.matureIndexs) - numel(this.complex.matureIndexs), 2) = foldingComplexs(j - 2*numel(this.monomer.matureIndexs)-numel(this.complex.matureIndexs), 2) + 1;
00642                 end
00643             end
00644             
00645             %update metabolites
00646             this.substrates(reactantIdxs) = this.substrates(reactantIdxs) - numFoldingProteins;
00647             this.substrates(productIdxs)  = this.substrates(productIdxs)  + numFoldingProteins;
00648             
00649             %update monomers
00650             this.monomers(this.monomer.matureIndexs, cIdxs) = ...
00651                 this.monomers(this.monomer.matureIndexs, cIdxs) + ...
00652                 foldingMonomers;
00653             this.monomers(this.monomer.misfoldedIndexs, cIdxs) = ...
00654                 this.monomers(this.monomer.misfoldedIndexs, cIdxs) - ...
00655                 foldingMonomers;
00656             
00657             %update complexs
00658             this.complexs(this.complex.matureIndexs, cIdxs) = ...
00659                 this.complexs(this.complex.matureIndexs, cIdxs) + ...
00660                 foldingComplexs;

00662                 this.complexs(this.complex.misfoldedIndexs, cIdxs) - ...
00663                 foldingComplexs;

00666         %Model complex degradation as a stochastic enzyme-dependent poisson
00667         %process. To simplify the modeling of complex degradation, this method
00668         %first stochastically selects cytosolic complexes to degrade, and then
00669         %breaks up the selected complexes into their constituents parts:
00670         %prosthetic groups, protein monomers, and RNAs, and finally relies on
00671         %the protein  monomer and RNA degradation models
00672         %(evolveState_DegradeMonomers and RNA Decay process, respectively) to
00673         %degrade the individual components. The method marks the component
00674         %protein monomers and RNAs and selected complexes as "damaged" (that is
00675         %in a state with 0 half life), so that the the protein monomer and RNA
00676         %machinery know to degrade these components.
00677         %
00678         %Note: only cytosolic complexes are degraded because the M. genitalium
00679         %protein degradation machinery (protease La) is a cytosolic protein, except that
00680                 %all "knocked" out proteins are degraded.
00681         function evolveState_DegradeComplexes(this)
00682             %compartment indices
00683             iC = this.compartment.cytosolIndexs;
00684             iTC = this.compartment.terminalOrganelleCytosolIndexs;
00685             
00686             %stochastically choose complexes to decay
00687             decayingRates = this.complexs .* ...
00688                 min(1e6, this.complex.decayRates(:, ones(1, this.compartment.count))) * this.stepSizeSec;
00689                         decayingRates(this.complex.decayRates < 1e6, [1:iC-1  iC+1:iTC-1  iTC+1:end]) = 0;
00690             decayingProteins = min(this.complexs, this.randStream.stochasticRound(decayingRates));
00691             decayingRates(decayingProteins == 0) = 0;
00692             if ~any(decayingProteins(:))
00693                 return;
00694             end
00695             decayedProteins = zeros(size(decayingProteins));
00696             
00697             %% decay unmodified complexes
00698             decayingSimpleProteins = decayingProteins(this.complexIndexs_unmodified, :);
00699             
00700             %decrement unmodified complexes
00701             if any(decayingSimpleProteins(:))
00702                 decayedProteins(this.complexIndexs_unmodified, :) = decayingSimpleProteins;
00703             end
00704             
00705             %% decay modified complexes up to limit of metabolites
00706             %(complex decay requires metabolites to reverse the affects of any
00707             %modifications)
00708             decayingModifiedRates = decayingRates(this.complexIndexs_modified, :);
00709             decayingModifiedProteins = decayingProteins(this.complexIndexs_modified, :);
00710             decayedModifiedProteins = zeros(size(decayingModifiedProteins));
00711             substrates = this.substrates;
00712             while any(decayingModifiedProteins(:))
00713                 %randomly select complex, weighted by copy number * decay rate
00714                 idx = this.randStream.randsample(numel(decayingModifiedRates), 1, true, decayingModifiedRates(:));
00715                 iProtein = this.complexIndexs_modified(mod(idx - 1, numel(this.complexIndexs_modified)) + 1);
00716                 
00717                 %check if sufficient resources available
00718                 if any(substrates < max(0, -this.complexDecayReactions(:, iProtein)))
00719                     break;
00720                 end
00721                 
00722                 %decrement complexes
00723                 decayingModifiedProteins(idx) = decayingModifiedProteins(idx) - 1;
00724                 decayedModifiedProteins(idx) = decayedModifiedProteins(idx) + 1;
00725                 if decayingModifiedProteins(idx) == 0
00726                     decayingModifiedRates(idx) = 0;
00727                 else
00728                     decayingModifiedRates(idx) = max(0, decayingModifiedRates(idx) - min(1e6, this.complex.decayRates(iProtein)) * this.stepSizeSec);
00729                 end
00730                 
00731                 %release prosthetic groups,
00732                 %invert any modifications
00733                 substrates = substrates + this.complexDecayReactions(:, iProtein);
00734             end
00735             decayedProteins(this.complexIndexs_modified, :) = decayedModifiedProteins;
00736             
00737             tmp = -decayedProteins;
00738             notUpdatingProteins = this.complex.updateExternalState(tmp, true);
00739             decayedProteins = decayedProteins - notUpdatingProteins;
00740             
00741             %decrease complexes. mark protein and RNA monomers for decay
00742             %(that is put them in a state with 0 half life) by
00743             %evolveState_DegradeMonomers and RNADecay
00744             this.complexs = this.complexs - decayedProteins;
00745             this.monomers(this.monomer.damagedIndexs, iC) = ...
00746                 this.monomers(this.monomer.damagedIndexs, iC) + ...
00747                 this.proteinComplexMonomerComposition * sum(decayedProteins, 2);
00748             this.RNAs(:, iC) = ...
00749                 this.RNAs(:, iC) + ...
00750                 this.proteinComplexRNAComposition * sum(decayedProteins, 2);
00751             this.substrates = ...
00752                 this.substrates + ...
00753                 this.complexDecayReactions * sum(decayedProteins, 2);
00754         end

00757         %available enzymes (metalloprotease FtsH and 6 peptidases) and their
00758         %kinetics (only of FtsH; peptidase kinetics haven't been characterized;
00759         %in this case we model that degradation can occur as only as at least 1

00777                 this.substrateIndexs_aminoAcids;

00780                 this.substrateIndexs_atp;
00781                 this.substrateIndexs_adp;
00782                 this.substrateIndexs_phosphate;
00783                 this.substrateIndexs_water;
00784                 this.substrateIndexs_hydrogen];

00804                 [0; peptidaseCosts] <= min(this.enzymes(this.enzymeIndexs_peptidases)) * this.stepSizeSec & ...

00822                     ProteinMonomer.computeBaseCount(abortedSeqs{i}, 22, 1:21, true),...

00851                 min(1e6, this.monomer.decayRates(:, ones(1, this.compartment.count))) * this.stepSizeSec;
00852                         decayingRates(this.monomer.decayRates < 1e6, [1:iC-1  iC+1:iTC-1  iTC+1:end]) = 0;

00854             decayingRates(decayingProteins == 0) = 0;
00855             if ~any(decayingProteins(:))
00856                 return;
00857             end
00858             decayedProteins = zeros(size(decayingProteins));
00859             
00860             iEnergy = [
00861                 this.substrateIndexs_atp
00862                 this.substrateIndexs_adp
00863                 this.substrateIndexs_phosphate
00864                 this.substrateIndexs_water
00865                 this.substrateIndexs_hydrogen
00866                 ];
00867             
00868             protease  = this.enzymes(this.enzymeIndexs_lonProtease) * this.stepSizeSec * this.lonProteaseSpecificRate;

00872             substrates = this.substrates;

00875                 idx = this.randStream.randsample(numel(decayingRates), 1, true, decayingRates(:));
00876                 iProtein = mod(idx - 1, size(this.monomers, 1)) + 1;
00877

00882                 proteaseCost = this.monomerLonProteaseCleavages(iProtein);

00885                         any(substrates(iEnergy) < max(0, substrateCost(iEnergy))) || ...

00887                         peptidase < 1

00896                 else
00897                     decayingRates(idx) = max(0, decayingRates(idx) - min(1e6, this.monomer.decayRates(iProtein)) * this.stepSizeSec);
00898                 end
00899
