00001 %Protein Folding
00002 %
00003 % @wholeCellModelID Process_ProteinFolding
00004 % @name             Protein Folding
00005 % @description
00006 %   Biology
00007 %   ==============
00008 %   Protein folding is the process whereby proteins which are produced as linear
00009 %   amino acid polymers relax to their most energetically favorable, and often
00010 %   more compact and catalytically active configuration. Some proteins relax
00011 %   quickly and spontaneously to their free energy minimum, while other proteins
00012 %   require chaperones to help them achieve their folded configuration more
00013 %   quickly. Additionally, some proteins coordinate prosthetic groups such as
00014 %   metal ions and other small molecules during this protein rearrangement
00015 %   process. Prosthetic groups may help stabilize the protein's catalytically
00016 %   active configuration, or may perform folding process.
00017 %
00018 %   In addition to protein chaperones, membrane protein folding may be assisted
00019 %   by phosphatidyl ethanolamine (PE) and phosphatidyl glycerol (PG) [PUB_0646].
00020 %   This effect is not currently modeled by this process.
00021 %
00022 %   This process simulates
00023 %   1. Prosthetic group complexation
00024 %      - Inorganic ions: Fe2, Fe3, K, Mg, Mn, Na, Zn
00025 %   2. Protein folding, where required mediated by ATP-dependent chaperone(s)
00026 %      [PUB_0014, PUB_0644]:
00027 %      a. Trigger factor (Tig, MG_238_MONOMER) – Highly expressed protein which
00028 %         possibly interacts with the L23 subunit of every ribosome,
00029 %         co-translationally assisting in the early folding every protein
00030 %         [PUB_0014, PUB_0644]. Binds vacant ribosomes with half-life of 11 s
00031 %         [PUB_0005]; binds ribosome-peptide complexes with half-life of 15-50 s
00032 %         [PUB_0005]; remains associated with the peptide for a up to more
00033 %         seconds after peptide release from the ribosome [PUB_0005]. Trigger
00034 %         factor-50S ribosome affinity: KD=1 ?M [PUB_0005]. We model trigger as
00035 %         required for the proper folding of all proteins.
00036 %      b. Chaperone (DnaK, MG_305_MONOMER) – Folds 5-18% of proteins [PUB_0014,
00037 %         PUB_0644]. Typically folds proteins > 30 kDa, typically in less than 2
00038 %         min [PUB_0014, PUB_0644]. Binds backbone of short, linear, unfolded,
00039 %         hydrophobic peptide segments [PUB_0014, PUB_0644]. ATP hydrolysis and
00040 %         peptide release catalyzed by GrpE (MG_201_DIMER) [PUB_0014, PUB_0644].
00041 %         Regulated by co-chaperone DnaJ (MG_019_DIMER) which interacts with the
00042 %         peptide side chains [PUB_0014, PUB_0644].
00043 %      c. Chaperonin (GroEL, GroES, MG_392_393_21MER) – Folds 10-15% proteins by
00044 %         ATP-dependent mechanism [PUB_0014, PUB_0644]. Typically folds proteins
00045 %         20-60 kDa in size [PUB_0014, PUB_0644] with half-life > 10 min
00046 %         [PUB_0014, PUB_0644], 30-60 s [PUB_0389].
00047 %
00048 %   Knowledge Base
00049 %   ==============
00050 %   The chaperones (except Tig, which all protein monomers require) and
00051 %   prosthetic groups required to fold each protein were reconstructed from the
00052 %   literature (see following sections), organized into the knowledge base, and
00053 %   are retrieved from the knowledge base in initializeConstants. Where known
00054 %   the stoichiometry of each prosthetic group was stored in the knowledge base;
00055 %   prosthetic groups with unreported stoichiometry are indicated with
00056 %   stoichiometry values of -1.
00057 %
00058 %   Chaperones
00059 %   ++++++++++++++
00060 %   The chaperone requirements for the folding of each protein were compiled
00061 %   for other bacteria (see sources below), and mapped to M. genitalium by
00062 %   homology.
00063 %
00064 %       Tig    may interact with all nascent peptides at the ribosome
00065 %              polypeptide exit site [PUB_0005, PUB_0009, PUB_0388] and assist
00066 %              in early folding. Binds vacant ribosomes with half-life of 11 s
00067 %              [PUB_0005]; binds ribosome-peptide complexes with half-life of
00068 %              15-50 s [PUB_0005]; remains associated with the peptide for a
00069 %              up to more seconds after peptide release from the ribosome
00070 %              [PUB_0005]. Trigger factor-50S ribosome affinity: KD=1 ?M
00071 %              [PUB_0005]. We model trigger as required for the proper folding
00072 %              of all proteins.
00073 %       DnaK   Deuerling et al performed a proteome-scale search for DnaK
00074 %              substrates in E. coli [PUB_0388]
00075 %       GroEL  Kerner et al performed a proteome-scale search for GroEL
00076 %              substrates in E. coli [PUB_0389] and Endo and Kurusu performed
00077 %              a proteome-scale search for GroEL substrates in B. subtilis
00078 %              [PUB_0391]
00079 %       FtsH   may act as molecule chaperone for membrane proteins
00080 %              [PUB_0014]. Several other functions have also been associated
00081 %              with FtsH. To date no proteome-scale studies of FtsH activity
00082 %              has been performed, and FtsH's chaperone substrates are not
00083 %              well characterized. Conequently we chose not to model FtsH as
00084 %              a molecular chaperone, but rather a protease.
00085 %
00086 %   Substrates of SecB, a molecular chaperone not present in M. genitalium, have
00087 %   also been identified on proteome-scale in E. coli [PUB_0390].
00088 %
00089 %   The reconstruction found that complexes typically require no chaperones to
00090 %   fold, with the notable exception of MG_392_393_21MER (which itself is a
00091 %   chaperone), which requires four.
00092 %
00093 %   Prosthetic groups
00094 %   ++++++++++++++
00095 %   Reaction coenzymes and prosthetic groups were reconstructed from several
00096 %   sources including the databases BioCyc [PUB_0006], BRENDA [PUB_0570],
00097 %   GenoBase [PUB_0386], Kinetikon [PUB_0571], Metal-MaCiE [PUB_0387], and
00098 %   UniProt [PUB_0096] and the primary literature [PUB_0131]. These sources
00099 %   frequently report protein prosthetic groups as a list of chemical species
00100 %   (eg. metal ions) which can each fill the same prosthetic group role. We
00101 %   simplified these ambiguous lists of possible prosthetic groups to a single
00102 %   chemical species by choosing the most common chemical species according to
00103 %   cell composition (Na+, K+ > Mg2+, Cl- > Fe2+, Fe3+ > Ca2+ > Mn2+ > Cu2+ >
00104 %   Mo6+, Zn2, Co2, Ni2 [PUB_0393, PUB_0394, PUB_0395]), or the species with the
00105 %   highest protein affinity (Irving-Williams Series, Mn2+ < Fe2+, Fe3+ < Co2+ <
00106 %   Ni2+ < Cu2+ > Zn2+ [PUB_0404]).
00107 %
00108 %   Roughly 20% of the monomers and 1% of the complexes bind at least one
00109 %   prosthetic group.
00110 %
00111 %   Representation
00112 %   ==============
00113 %   substrates, enzymes, unfoldedMonomers, unfoldedComplexs, foldedMonomers, and
00114 %   foldedComplexs represent the counts of free metabolites, chaperones,
00115 %   unfolded protein monomers, unfolded protein complexes, folded protein
00116 %   monomers, and folded protein complexes. Each of these component types have
00117 %   been mapped from the several compartments of the simulation class to one
00118 %   pseudo compartment in this process. That is this process only "sees" the
00119 %   counts of metabolites, chaperones, and protein monomers and complexes that
00120 %   are relevant to protein folding. The process contains no intermediate
00121 %   representation of protein folding, that is protein folding is simulated as a
00122 %   all-or-nothing process that either does not occur or proceeds to completion
00123 %   within a single time step.
00124 %
00125 %   The proteinChaperoneMatrix and proteinProstheticGroupMatrix are adjacency
00126 %   matrices which represent the chaperone and prosthetic groups each protein
00127 %   monomer and complex requires to achieve its catalytically active
00128 %   configuration. Where the stoichiometry of a prosthetic group is unknown and
00129 %   indicated in the knowledge base with the value -1, here we assume a
00130 %   stoichiometry of 1 and set the element of proteinProstheticGroupMatrix
00131 %   accordingly.
00132 %
00133 %   Initialization
00134 %   ==============
00135 %   All protein monomers and complexes are initialized to the mature state. This
00136 %   is accomplished by the simulation class initializeState method.
00137 %
00138 %   Simulation
00139 %   ==============
00140 %   As with the other protein maturation processes, protein folding is modeled as
00141 %   as all-or-nothing process at the single time-step level. A protein only
00142 %   proceeds through this phase if its chaperones (if any) and any prosthetic
00143 %   group ions are all present in the same time step. Chaperone kinetics are not
00144 %   well characterized, and are not currrently modeled. Proteins that require
00145 %   neither prosthetic groups nor chaperones are considered folded after one
00146 %   time step.
00147 %
00148 %   Algorithm
00149 %   ++++++++++++++
00150 %   While(true)
00151 %     1. Calculate numbers of proteins that can fold based on prosthetic group,
00152 %        chaperone, and unfolded protein availability.
00153 %     2. Randomly select proteins to fold weighted by limits calculated in step
00154 %        (1).
00155 %     3. Update counts of prosthetic groups, chaperones, and unfolded and folded
00156 %        proteins.
00157 %     4. Repeat until insufficient resources to further fold proteins
00158 %   End
