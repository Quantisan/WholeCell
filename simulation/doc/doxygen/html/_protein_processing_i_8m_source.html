00003 % @wholeCellModelID Process_ProteinProcessingI
00004 % @name             Protein Processing I
00005 % @description
00006 %   Biology
00007 %   ===========
00008 %   Following translation, nascent peptides are deformylated, cleaved,
00009 %   translocated, folded, and modified. First, peptide deformylase (MG_106)
00010 %   deformylates the N-terminal formylmethionine of each nascent peptide.
00011 %   Second, methionine aminopeptidase (MG_172) cleaves the N-terminal methionine
00012 %   of 35 peptides. Second (see protein translocation process), 117 integral
00013 %   membrane, lipoproteins, and extracellular proteins bind the SecA translocase
00014 %   and are translocated into and through the plasma membrane via the
00015 %   SecYEGDF-YidC pore. Third (see protein processing II process), diacylglyceryl
00016 %   is transferred to the C-terminal cysteine of the signal sequence of each
00017 %   lipoprotein by diacylglyceryl transferase, and the signal sequence of each
00018 %   lipoprotein is cleaved by signal peptidase II. Next (see protein
00019 %   folding process), 85 peptides bind inorganic ions at particular sites
00020 %   and 64 peptides fold with the assistance of the chaperones and chaperonins
00021 %   DnaJ, DnaK, GroEL, GroES, and GrpE. All peptides require trigger factor
00022 %   to properly fold. Finally (see protein modification process), 20 peptide
00023 %   species are modified at 63 sites by 3 enzymes – serine/threonine protein
00024 %   kinase, lipoate ligase, and alpha glutamate ligase.
00025 %
00026 %   This process simulates
00027 %   - N-terminal peptide deformylation, and
00028 %   - N-terminal amino acid cleavage.
00029 %
00030 %   Knowledge Base
00031 %   ===========
00032 %   Every M. genitalium protein requires deformylation, and roughly 7%
00033 %   require N-terminal amino acid cleavage. N-terminal methionine cleavages were
00034 %   reconstructed by mapping N-terminal methionine cleavages observed in
00035 %   Shewanella oneidensis MR-1 [PUB_0280] onto homologous M. genitalium genes.
00036 %   The N-terminal methionine cleavage state of each protein monomer is
00037 %   organized in the knowledge base, and loaded into the
00038 %   nascentMonomerNTerminalMethionineCleavages property of this process by the
00039 %   initializeConstants method.
00040 %
00041 %   Representation
00042 %   ===========
00043 %   substrates, enzymes, unprocessedMonomers, and processedMonomers represent
00044 %   the counts of metabolites, the deformylase and methionine aminopeptidase,
00045 %   and nascent and deformylated, cleaved protein monomers. The compartment
00046 %   dimension of each of these properties has length 1. That is the process only
00047 %   accesses the counts of these objects in the relevant compartments. This
00048 %   process doesn't represent any additional intermediate processed states. This
00049 %   process treats N-terminal deformylation and methionine cleavage as an
00050 %   all-or-nothing event that either proceeds to complete within a single time
00051 %   step, or does not occur at all.
00052 %
00053 %   nascentMonomerNTerminalMethionineCleavages is a boolean which represents
00054 %   whether or not the N-terminal methionine of each protein monomer must be
00055 %   cleaved.
00056 %
00057 %   Initialization
00058 %   ===============
00059 %   All protein monomers are initialized to the mature state. This is
00060 %   implemented by the simulation class initializeState method.
00061 %
00062 %   Simulation
00063 %   ===========
00064 %   1. Compute the maximum number of peptides that can be processed based on the
00065 %      availability of the two enzymes.
00066 %   2. Randomly select peptides to be processed, weighted by the counts of each
00067 %      protein species.
00068 %   3. Update the counts of unprocessed and processed proteins monomers.
00069 %      Decrement the counts of available enzyme activity.
00070 %   4. Compute the maximum number of peptides which don't require cleavage that
00071 %      can be processed based on the availability of peptide deformylase
00072 %      activity.
00073 %   5. Randomly select peptides to be deformylated, weighted by the counts of
00074 %      each protein species.
00075 %   6. Update the counts of unprocessed and processed proteins monomers.
00076 %      Decrement the counts of available deformylase activity.
00077 %
