00002 %
00003 % @wholeCellModelID Process_ProteinProcessingII
00004 % @name             Protein Processing II
00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   Proteins are produced in the cytoplasm, and integral membrane proteins,
00009 %   lipoproteins, and extracellular proteins must be translocated into and
00010 %   through the cell membrane to reach their intended destination. To be
00011 %   recognized by the translocation machinery lipoprotein and secreted proteins
00012 %   contain type II N-terminal signal sequences, which are typically 10-15 amino
00013 %   acids long and positively charged. Following translocation lipoproteins must
00014 %   be anchored to the outter leaflet of the cell membrane by the addition of
00015 %   diacylglyceryl by diacylglyceryl transferase (Lgt, MG_086) to what will
00016 %   become the C-terminal cysteine, and their signal sequence must be cleaved by
00017 %   signal peptidase II (LspA, MG_210) at lipoboxes (L[ASI][GA]C). M. genitalium
00018 %   does not contain an apolipoprotein transacylase [PUB_0655, PUB_0656,
00019 %   PUB_0657]. The signal peptides of secreted proteins is similarly cleaved.
00020 %
00021 %   This process simulates lipoprotein and secreted protein maturation:
00022 %   - diacylglyceryl transfer – Lgt (MG_086) catalyzes transfer of
00023 %     diacylglycerol group to sulfhydryl group of lipobox cysteine
00024 %     [PUB_0654].
00025 %   - signal peptide cleavage – LspA (MG_210) cleaves lipoprotein at
00026 %     lipobox cysteine [PUB_0654].
00027 %
00028 %   Knowledge Base
00029 %   ===============
00030 %   The localization of each protein, and the signal peptide type and length of
00031 %   lipoproteins and secreted proteins was compiled from several sources (see
00032 %   protein translocation process). The information was organized in the
00033 %   knowledge base, and is the type of each signal peptide is encoded in the
00034 %   this process's lipoproteinMonomerIndexs, secretedMonomerIndexs, and
00035 %   unprocessedMonomerIndexs properties by initializeConstants.
00036 %
00037 %   Representation
00038 %   ===============
00039 %   The substrates and enzymes represents the counts of available metabolites
00040 %   and diacylglyceryl transferase and signal peptidase enzymes.
00041 %   unprocessedMonomers, processedMonomers, and signalSequenceMonomers represent
00042 %   the counts of unanchored, uncleaved protein monomers; anchored, cleaved
00043 %   protein monomers; and the separted, free signal sequences. This process does
00044 %   not represent any intermediate states in the anchoring and cleavage of lipo-
00045 %   and secreted proteins. This process consideres anchoring and cleavage to be
00046 %   an all-or-nothing event.
00047 %
00048 %   The lipoproteinMonomerIndexs and secretedMonomerIndexs properties represent
00049 %   the indices of lipo-and secreted proteins and their released signal
00050 %   sequences within unprocessedMonomers, processedMonomers, and
00051 %   signalSequenceMonomers. unprocessedMonomerIndexs indicating the indices of
00052 %   non-lipo-, non-secreted proteins and their released signal sequences (not
00053 %   used in simulation; only allocated for convenience and parallelism) within
00054 %   unprocessedMonomers, processedMonomers, and signalSequenceMonomers.
00055 %
00056 %   Initialization
00057 %   ===============
00058 %   All protein monomers are initialized to the mature state. This is
00059 %   accomplished by the simulation class initializeState method.
00060 %
00061 %   Simulation
00062 %   ===============
00063 %   1. Compute the maximum number of peptides that can be processed based on the
00064 %      availability of metabolites, and of the two enzymes.
00065 %   2. Randomly select peptides to be processed, weighted by the counts of each
00066 %      protein species.
00067 %   3. Update the counts of unprocessed and processed proteins monomers.
00068 %      Decrement the counts of available metabolites and enzyme activity.
00069 %   4. Compute the maximum number of peptides which don't require anchoring that
00070 %      can be processed based on the availability of signal peptidase II
00071 %      activity.
00072 %   5. Randomly select peptides to be cleaved, weighted by the counts of
00073 %      each protein species.
00074 %   6. Update the counts of unprocessed and processed proteins monomers.
00075 %      Decrement the counts of available signal peptidase II activity.
00076 %   7. Transition proteins which don't requiring anchoring or cleavage (eg.
00077 %      cytosolic and integral membrane proteins). That is set processedMonomers
00078 %      equal to its sum with unprocessedMonomers for these monomers, and set
00079 %      unprocessedMonomers to zero for these monomers.
