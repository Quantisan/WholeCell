00002 %
00003 % @wholeCellModelID Process_RNADecay
00004 % @name             RNA Decay
00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   In presence of ribonucleases such as ribonuclease R (MG_104_MONOMER) RNAs
00009 %   have relatively short half lives compared to that of other macromolecules
00010 %   (eg. protein, DNA) and the M. genitalium cell cycle length. The relatively
00011 %   short half lives of RNAs enables the small M. genitalium with its very small
00012 %   pool of RNAs and particularly mRNAs to sample a broader range of
00013 %   configurations of the RNA pool over a shorter period that would be possible
00014 %   with longer half lifes. This helps the cell more finely tune the expression
00015 %   of proteins, more efficiently execute cell-cycle dependent events, and
00016 %   respond to the external environment. This enhanced fitness due to short RNA
00017 %   half lifes comes at a large energetic cost however.
00018 %
00019 %   In addition to ribonucleases, aminoacylated RNAs require peptidyl tRNA
00020 %   hydrolase (MG_083_MONOMER) to release their conjugated amino acids.
00021 %
00022 %   This process decays all species of RNA, and at all maturation states
00023 %   including aminoacylated states.
00024 %
00025 %   Knowledge Base
00026 %   ===============
00027 %   The knowledge base contains experimentally measured half lifes of many RNA
00028 %   species measured largely in E. coli and mapped to M. genitalium by homology.
00029 %   These half lifes are refined, by simulation.fitConstants to make them
00030 %   consistent with other experimental data used to fit the model. Prior to
00031 %   fitting missing half lifes are imputed either as the average of that of all
00032 %   measured RNA species.
00033 %
00034 %      Type   Avg Half Life (m)
00035 %      ====   =================
00036 %      mRNA   4.5 +/- 2.0
00037 %      rRNA   150
00038 %      sRNA   89
00039 %      tRNA   45
00040 %
00041 %   Representation
00042 %   ===============
00043 %   The substrates, enzymes, and RNAs properties represent the counts of
00044 %   metabolites, ribonuclease R and peptidyl tRNA hydrolase enzymes, and RNAs.
00045 %   This process contains no intermediate representation of RNA degradation; RNA
00046 %   degradation is treated as an all-or-nothing event that either proceeds to
00047 %   complete with a time step or doesn't progress at all.
00048 %
00049 %   decayRates represents the decay rate of each RNA species in seconds.
00050 %   decayRates is informed by experimentally measured RNA half lifes organized
00051 %   in the knowledge base, and fit by simulation.fitConstants. decayReactions
00052 %   represents the metabolites required to decay each RNA species, and the
00053 %   metabolic byproducts of the decay of each RNA species. decayReactions is
00054 %   computed by the knowledge RNA classes based on the sequence, processing, and
00055 %   modifications of each RNA species.
00056 %
00057 %   Initialization
00058 %   ===============
00059 %   All RNAs are initialized to the mature state. This is accomplished by the
00060 %   simulation class initializeState method.
00061 %
00062 %   Simulation
00063 %   ===============
00064 %   This process models RNA decay as an enyzme-dependent poisson process with
00065 %   rate parameter:
00066 %     lambda = RNAs .* decayRates * stepSizeSec
00067 %
00068 %   Algorithm
00069 %   +++++++++++++++
00070 %   1. Stochastically select RNAs to decay based on poission distribution with
00071 %      lambda = RNAs .* decayRates * stepSizeSec
00072 %   2. (Ignore limits to decay posed by availability of metabolite reactants
00073 %      since the only reactant is water, and water is abundantly available)
00074 %   3. Limit RNA decay by available enzyme activity
00075 %      a. All RNAs require ribonuclease R to decay
00076 %      b. Additionally, only decay aminoacylated tRNAs up to the limit of
00077 %         available peptidyl tRNA hydrolase activity.
00078 %   4. Update counts of RNAs
00079 %   5. Update counts of metabolic byproducts of RNA decay
00080 %

00299             import edu.stanford.covert.cell.kb.ssRNA;

00314             abortedSeqs = this.transcripts.abortedSequences;
00315             abortedTfs = false(size(abortedSeqs));

00317                 substrateCost = ssRNA.computeDecayReaction(ssRNA.computeBaseCount(...

00319                     numel(abortedSeqs{i}), 'linear', ...
00320                     this.substrateIndexs_water, this.substrateIndexs_hydrogen)';
00321                 if any(this.substrates < -substrateCost)
00322                     break;
00323                 end
00324                 abortedTfs(i) = true;
00325                 this.substrates = this.substrates + substrateCost;
00326             end
00327             this.transcripts.abortedTranscripts = this.transcripts.abortedTranscripts(~abortedTfs, :);
00328             
00329             %% Stochastically decay free RNA as poisson process
00330             decayingRNAs = min(this.randStream.random('poisson', ...
00331                 this.RNAs .* min(1e6, this.rna.decayRates * this.stepSizeSec)), ...
00332                 this.RNAs);
00333             if ~any(decayingRNAs)
00334                 return;
00335             end
00336             
00337             %Require peptidyl tRNA hydrolase to decay aminoacylated tRNAs
00338             tmp = decayingRNAs(this.rna.aminoacylatedIndexs);
00339             tmp2 = zeros(size(tmp));
00340             while any(tmp)
00341                 if peptidylTRNAHydrolase <= 0
00342                     break;
00343                 end
00344                 idx = this.randStream.randsample(numel(tmp), 1, true, tmp);
00345                 tmp(idx) = tmp(idx) - 1;
00346                 tmp2(idx) = tmp2(idx) + 1;
00347                 peptidylTRNAHydrolase = peptidylTRNAHydrolase - 1;
00348             end
00349             decayingRNAs(this.rna.aminoacylatedIndexs) = tmp2;
00350             
00351             %require substrates (water) to decay RNAs
00352             tmp = decayingRNAs;
00353             tmp2 = zeros(size(tmp));
00354             water = this.substrates(this.substrateIndexs_water);
00355             waterReqs = max(0, -this.decayReactions(:, this.substrateIndexs_water));
00356             while any(tmp)
00357                 idx = this.randStream.randsample(numel(tmp), 1, true, tmp);
00358                 if water < waterReqs(idx);
00359                     break;
00360                 end
00361                 water = water - waterReqs(idx);
00362                 tmp(idx) = tmp(idx) - 1;
00363                 tmp2(idx) = tmp2(idx) + 1;
