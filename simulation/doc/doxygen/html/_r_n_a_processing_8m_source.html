00003 % @wholeCellModelID Process_RNAProcessing
00004 % @name             RNA Processing
00005 % @description
00006 %   Biology
00007 %   ==================
00008 %   Transcription produces mono- as well as polycistronic RNAs.
00009 %   Following transcription polycistronic rRNA, tRNA, and tmRNA transcripts are
00010 %   cleaved, modified, and aminoacylated. First, 45S rRNA transcripts are
00011 %   cleaved by ribonuclease III (MG_367_DIMER) into 5S, 16S, and 23S rRNA
00012 %   precursors [PUB_0038, PUB_0039]. Next these precursors are cleaved at their
00013 %   3' and 5' ends by ribonuclease J and RsgA [PUB_0038]. Similarly the 5' ends
00014 %   of each tRNA and tmRNA species are cleaved by ribonuclease P [PUB_0039,
00015 %   PUB_0649]. Ribonuclease III also cleaves scRNA at two sites [PUB_0039].
00016 %   Second, 13 enzymes formylate, lysidinate, methylate, pseudouridylate, and
00017 %   thiolate 86 specific rRNA and tRNA bases. Third 19 tRNA synthetases
00018 %   conjugate 19 amino acids to 37 tRNA and tmRNA species, and 2 tRNA
00019 %   transferases complete the aminoacylation of the formylmethionine and
00020 %   glutamine tRNAs.
00021 %
00022 %   rRNA, tRNA, and tmRNA cleavage, modification, and aminoacylation are modeled
00023 %   motivated by mass-action kinetics. First, the maximum rate of maturation of
00024 %   each RNA species is calculated based on (1) the abundance of immature
00025 %   transcripts, (2) the abundance of substrates and enzymes required to cleave
00026 %   or modify each RNA species, and (3) the experimentally measured kinetic rate
00027 %   of each enzyme required to cleave or modify each RNA species. Second, RNA
00028 %   maturations are randomly selected according to the calculated maximum rates.
00029 %   Finally, steps (1) and (2) are repeated until insufficient resources are
00030 %   available to mature additional RNAs.
00031 %
00032 %   This process simulates the cleavage of polycistronic r/s/t RNA transcripts
00033 %   into individual genes:
00034 %   - 30S RNA        -> pre 16/17S RNA, p23S RNA, 9S RNA    RNAseIII (MG_367)
00035 %   - pre 16/17S RNA -> 16S RNA                             5' end:rnjA (MG_139); 3' end:rsgA (MG_110)
00036 %   - p23S RNA       -> 23S RNA                             5' end:rnjA (MG_139), deaD (MG_425)
00037 %   - 9S RNA         ->  5S RNA
00038 %   - 3' end tRNA precursors                                RNAseIII (MG_367)
00039 %   - 5' end tRNA precursors                                RNaseP (MG_0003, MG_465)
00040 %   - scRNA (MG_0001) precusors                             RNAseIII (MG_367)
00041 %   - 5' end tmRNA precursors                               RNaseP (MG_0003, MG_465)
00042 %
00043 %   Reactions
00044 %   ++++++++++++++++++
00045 %   DeaD           phosphorolytic                      3.6.4.13
00046 %   RsgA           phosphorolytic                      3.6.5.2, 3.6.1.-
00047 %   RNAaseIII      hydrolytic       endoribonuclease   3.1.26.3
00048 %   RNAseP         hydrolytic       endoribonuclease   3.1.26.5
00049 %   RNAseJ         hydrolytic       endoribonuclease   3.1.27.2
00050 %
00051 %   Cofactors
00052 %   ++++++++++++++++++
00053 %   DeaD           Mg2+                [PUB_0071]
00054 %   RsgA       (1) Zn2+                [PUB_0096]
00055 %   RNAaseIII      Mg2+                [PUB_0039]
00056 %   RNAseJ     (2) Zn2+                [PUB_0096]
00057 %   RNAseP     (3) Mg2+ or (3) Mn2+    [PUB_0012, PUB_0013, PUB_0043]
00058 %
00059 %   Kinetics:
00060 %   ++++++++++++++++++
00061 %   DeaD       1.48   1/s   B. subtilis  [PUB_0688]
00062 %   RsgA       0.2917 1/s   E. coli      [PUB_0103]
00063 %   RNAseP     0.027  1/s   B. subtilis  [PUB_0013]
00064 %   RNAseP     6      1/s   B. subtilis  [PUB_0690]
00065 %   RNAseIII   7.7    1/s   E. coli      [PUB_0691]

00071 %   RsgA       1     GTP  1/rxn   E. coli      [PUB_0103]

00073 %   Knowledge Base
00074 %   ==================
00075 %   The transcription unit organization was predicted by mapping the
00076 %   experimentally determined genome organization of M. pnemoniae by Güell et al
00077 %   [Table S5 "Suboperons", PUB_0418] onto the M. genitalium genome by homology
00078 %   with 4 modifications:
00079 %   - r/s/tRNAs were organized into transcription units according to their
00080 %     "reference operons" (table s4)
00081 %   - All mRNAs Güell Serrano et al did not assign to a suboperon because of
00082 %     insufficient evidence (low quality expression data), were assigned to
00083 %     their own transcription units. None of these mRNAs are located within
00084 %     suboperons.
00085 %   - All 5 mRNA genes which don't have homologs in M. pneumoniae were assigned
00086 %     to their own transcription units. These occur outside other transcription
00087 %     units.
00088 %   - Genes which have undergone rearrangements between M. pnuemoniae and M.
00089 %     genitalium were assigned to their own transcription units
00090 %
00091 %   The promoter (-35 and -10 boxes and TSS) for each transcription unit was set
00092 %   predicted for its 3' gene by Weiner et al [PUB_0411].
00093 %
00094 %   The composition of each transcription unit, and the location of its -35 and
00095 %   -10 boxes and TSS relative to the start coordinate of the 3' gene was stored
00096 %   in the knowledge base, and it used by the TranscriptionUnit and Gene classes
00097 %   to compute the sequences of transcription unit, gene, and intergenic
00098 %   segment.
00099 %
00100 %   The kinetics and energy requirement of each RNA processing enzyme are
00101 %   organized as parameters in the knowledge base. These values were curated
00102 %   several papers [PUB_0069, PUB_0103, PUB_0688, PUB_0690, PUB_0691, PUB_0692].
00103 %
00104 %   Representation
00105 %   ==================
00106 %   substrates and enzymes represent the counts of metabolites and RNA
00107 %   processing enzymes. unprocessedRNAs, processedRNAs, and intergenicRNAs
00108 %   represent counts of RNAs. unprocessedRNAs represents the counts of nascent
00109 %   RNAs produced by the RNA polymerase. processedRNA represents the counts of
00110 %   r/s/tRNA precursors produced from cleavage of nascent transcripts, and mRNAs
00111 %   that remain joined as transcription units. intergenic RNAs represents the
00112 %   counts of RNA segments between r/s/tRNA genes in transcripts that are
00113 %   released during the processing of these transcripts.
00114 %
00115 %   enzymeSpecificRate_* are the kcats of the five RNA processing enzymes. We
00116 %   assume that each enzyme has the same kcat for all reactions it catalyzes.
00117 %   enzymeEnergyCost_* are the amount of ATP required by the phosphorolytic RNA
00118 %   processing enzymes (DeaD and Rsga) per cleavage reaction.
00119 %
00120 %   rna.nascentRNAMatureRNAComposition, rna.intergenicRNAMatrix,
00121 %   reactantByproductMatrix, and catalysisMatrix are adjacency matrices.
00122 %   nascentRNAMatureRNAComposition represents the the processed RNAs that arise
00123 %   from each unprocessed RNA (eg. the r/s/t RNAs that arise from transcripts
00124 %   containing multiple genes). intergenicRNAMatrix represents the intergenic
00125 %   RNA segments that arise from the cleavage of r/s/tRNA transcripts into their
00126 %   individual genes. reactantByproductMatrix represents the amounts of
00127 %   metabolites required to cleave each polycistronic transcript, and the
00128 %   metabolic byproducts of their cleavage. catalysisMatrix represents the
00129 %   amount of enzyme required to cleave each polycistronic transcript.
00130 %
00131 %   Initialization
00132 %   ==================
00133 %   All RNAs are initialized to the mature state. This is accomplished by the
00134 %   simulation class initializeState method.
00135 %
00136 %   Simulation
00137 %   ==================
00138 %   For each RNA class (mRNA, rRNA, sRNA, tRNA) in a random order:
00139 %   1. Determine maximum number of RNAs that can be processed based on available
00140 %      - RNA processing enzymes and their kinetics
00141 %      - free metabolites
00142 %      - RNAs that need processing
00143 %   2. Randomly select among unprocessed RNAs proportional to count
00144 %   3. Update state:
00145 %      a. Decrement unprocessed RNAs
00146 %      b. Increment processed RNAs, intergenic RNAs
00147 %      c. Decrement metabolic reactants
00148 %      d. Increment metabolic byproducts%
