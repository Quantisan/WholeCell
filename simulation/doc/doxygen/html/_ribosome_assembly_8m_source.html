00002 %
00003 % @wholeCellModelID Process_RibosomeAssembly
00004 % @name             Ribosomal Assembly
00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   Assembly of the 30S and 50S ribosomal particles is a special case of
00009 %   macromolecular complexation in which subunits are incorporated in a
00010 %   stereotyped pattern [PUB_0660, PUB_0661], and which requires six
00011 %   energy-dependent GTPases – EngA, EngB, Era, Obg, RbfA, and RbgA. The
00012 %   exact energy requirement of each GTPase is unknown. Here we assume that
00013 %   each GTPase that has been reported to be required to form each
00014 %   ribosomal particle requires 1 GTP per particle. 
00015 %
00016 %   Knowledge Base
00017 %   ===============
00018 %   The 30S and 50S ribosomal particles are represented as complexes in the
00019 %   knowledge base. Their RNA and protein subunit composition was curated from
00020 %   the literature, and is stored in association with the complexes in the
00021 %   knowledge base. This composition is loaded by initializeConstants into this
00022 %   process's proteinComplexRNAComposition and proteinComplexMonomerComposition
00023 %   properties.
00024 %
00025 %   Representation
00026 %   ===============
00027 %   substrates, enzymes, RNAs, monomers, and complexs represent the counts of
00028 %   free metabolites (eg. ATP, ADP, Pi, H2O, H+), the ribosomal assembly
00029 %   GTPases, the amounts of RNA and protein monomer ribosomal subunits, and the
00030 %   30S and 50S ribosomal particles. The process doesn't represent any
00031 %   intermediate state of ribosomal particle assembly; ribosomal particle
00032 %   assembly is assumed to be an all-or-nothing process on the time scale of
00033 %   this process. That is, we make the simplifying assumption that either a
00034 %   ribosomal particle completely forms within a single time step, or no
00035 %   progress in assembly of that particle is made during that time step.
00036 %
00037 %   proteinComplexRNAComposition and proteinComplexMonomerComposition represent
00038 %   the RNA and protein monomer composition of the 30S and 50S ribosomal
00039 %   subunits. complexationCatalysisMatrix represents the GTPases required to
00040 %   form each ribosomal particle.
00041 %
00042 %   Initialization
00043 %   ===============
00044 %   The process is initialized to a state with the maximal number of formed
00045 %   ribosomal particles given the amounts of initialized RNA and protein
00046 %   monomers. That is, the process is initialized to a state where insufficient
00047 %   RNA and protein monomers are available to form additional ribosomal subunits.
00048 %
00049 %   Simulation
00050 %   ===============
00051 %   In a randomized order over particles, for each ribosomal particle:
00052 %   1. Calculate the maximum number of particles that can form based on
00053 %      available RNA and protein monomer subunits, GTPases, and GTP.
00054 %   2. Increment the number of ribosomal particles. Decrement the numbers of RNA
00055 %      and protein monomer subunits, and GTP and water. Increment the counts of
00056 %      the byproducts of GTP hydrolysis (GDP, Pi, H).
00057 %
00058 %   This makes the simplying assumption that ribosomal assembly is fast compared
00059 %   to the 1s time scale of this process, and is energetically favorable such
00060 %   that in the presence of saturating enzymes and energy, ribosomal assembly is
00061 %   limited by the subunit availability.
00062 %
00063 %   References
00064 %   ===============
00065 %   1. Nierhaus KH (1991). The assembly of prokaryotic ribosomes.
00066 %      Biochimie. 76(3):739-55.[PUB_0660]
00067 %   2. Culver GM (2003). Assembly of the 30S ribosomal subunit.
00068 %      Biopolymers. 68(2):234-49. [PUB_0661]
00069 %
00070 % Author: Jonathan Karr, jkarr@stanford.edu
00071 % Affilitation: Covert Lab, Department of Bioengineering, Stanford University

00298         function evolveState(this)
00299             %stop if no GTP
00300             if ~this.substrates(this.substrateIndexs_gtp)
00301                 return;
00302             end
00303             
00304             %randomize order in which to try forming complexes
00305             randOrder = this.randStream.randperm(numel(this.complexWholeCellModelIDs));
00306             for j = 1:numel(this.complexWholeCellModelIDs)
00307                 i = randOrder(j);
00308 
00309                 %number of GTPs needed for GTPases to form complex
00310                 gtpPerComplex = this.getGtpPerComplex(i);
00311 
00312                 %number of complexes that can form, limited by:
00313                 %- enzymes
00314                 %- GTP
00315                 %- RNA subunits
00316                 %- protein monomer subunits
00317                 %
00318                 %assumes stoichiometry of each subunit is 1; this is verified in
00319                 %initializeConstants
00320                 newComplexs = floor(min([
00321                     this.substrates(this.substrateIndexs_gtp) / gtpPerComplex;
00322                     this.substrates(this.substrateIndexs_water) / gtpPerComplex;
00323                     this.RNAs(this.proteinComplexRNAComposition(:,i));
00324                     this.monomers(this.proteinComplexMonomerComposition(:,i))]));
00325                 if newComplexs == 0 || ~all(this.enzymes(this.complexationCatalysisMatrix(:,i)))
00326                     continue;
00327                 end
00328 
00329                 %update counts of complexes, RNAs, protein monomers, metabolites
00330                 this.complexs(i) = this.complexs(i) + newComplexs;
00331 
00332                 this.RNAs     = this.RNAs     - this.proteinComplexRNAComposition(:,i)     * newComplexs;
00333                 this.monomers = this.monomers - this.proteinComplexMonomerComposition(:,i) * newComplexs;
00334 
00335                 this.substrates = this.substrates + [-1; 1; 1; -1; 1] * newComplexs * gtpPerComplex; %substrate order is [GTP, GDP, Pi, H2O, H]
