00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   The terminal organelle is a 300 x 80 nm membrane-bound bleb believed to
00009 %   be involved in several processes including motility, adhesion,
00010 %   replication, and cytokinesis. The terminal organelle is electron lucent 
00011 %   and non-fibrillar and enriched for adhesins and adhesin accessory proteins 
00012 %   which help the adhesins localize to the terminal organelle. The terminal 
00013 %   organelle has been investigated in several studies by Mitchell Balish at
00014 %   Miami University of Ohio and Duncan Krause of the University of Georgia. In
00015 %   particular, they have examined the composition and assembly of the electron
00016 %   dense (protein) portion of the terminal organelle, and have examined the
00017 %   duplication and migration to the opposite pole of the daughter terminal
00018 %   organelle during cell division. Pich et al have studied the regulation of
00019 %   terminal organelle duplication.
00020 %
00021 %   The terminal organelle is composed of 8 proteins.
00022 %   - HMW1-3
00023 %   - MgPa
00024 %   - P32, P65, P110, P200
00025 %
00026 %   These 8 proteins assemble in a stereotyped order; that is, there is a
00027 %   heirarchical pattern to the assembly of the terminal organelle: several
00028 %   proteins require other proteins to be already localized in the terminal
00029 %   organelle to be incorporated into the terminal organelle, and two proteins
00030 %   HMW1 and HMW2 require the other either to already been incorporated in the
00031 %   terminal organelle, or free in the cytoplasm to be incorporated into the
00032 %   terminal organelle. No kinetic information has been reported on terminal
00033 %   organelle assembly.
00034 %
00035 %   During replication a second organelle is believed to duplicate from the
00036 %   first organelle and migrate toward the opposite pole.
00037 %
00038 %   Knowledge Base
00039 %   ===============
00040 %   The hierarchical pattern of terminal organelle assembly among the 8 proteins
00041 %   is encoded in 10 reactions in the knowledge base.
00042 %
00043 %   Representation
00044 %   ===============
00045 %   Proteins are incorporated according to one of two patterns:
00046 %   - cytoplasm -> terminal organelle cytoplasm
00047 %   - membrane  -> terminal organelle membrane
00048 %
00049 %   The counts of terminal organelle proteins are represented by the substrates
00050 %   property. Because proteins are only incorporated in two patterns, the
00051 %   substrates property has two compartments: one to represent the
00052 %   unincorporated compartment of each protein (cytoplasm/membrane) and a
00053 %   second for the incorporated compartment of each protein (terminal organelle
00054 %   cytoplasm/membrane).
00055 %
00056 %   The hierarchical pattern of terminal organelle assembly is represented in
00057 %   this process by localizationReactions and localizationSubstrates.
00058 %   localizationSubstrates represents the protein translocated by each of the
00059 %   10 reactions. localizationReactions represents the proteins (and the
00060 %   compartment in which the protein must be located) required for each
00061 %   translocation reaction to proceed.
00062 %
00063 %   Initialization
00064 %   ===============
00065 %   Terminal organelle proteins are all initialized into the terminal organelle
00066 %   compartments.
00067 %
00068 %   Simulation
00069 %   ===============
00070 %     1. Compute which reactions can proceed based on the amounts of
00071 %        unincorporated and incorporated proteins.
00072 %     2. Compute which proteins can localize to the terminal organelle based on
00073 %        step (1).
00074 %     3. Incorporate the proteins that can localize into the terminal organelle.
00075 %     4. Repeat steps 1-3 until no additional proteins can localize.
00076 %

00316         function evolveState(this)
00317             % incorporate proteins into terminal organelle
00318             % cytoplasm -> terminal organelle cytoplasma
00319             % membrane  -> terminal organelle membrane
00320             while any(this.substrates(:, this.compartmentIndexs_unincorporated))
00321                 activeReactions = ...
00322                     this.localizationReactions(:, :, 1) * (this.substrates(:, 1) > 0) + ...
00323                     this.localizationReactions(:, :, 2) * (this.substrates(:, 2) > 0) >= ...
00324                     this.localizationThreshold;
00325                 
00326                 localizingProteins = logical(...
00327                     this.substrates(:, this.compartmentIndexs_unincorporated) .* ...
00328                     (this.localizationSubstrates' * activeReactions));
00329                 if ~any(localizingProteins)
00330                     break; 
00331                 end
00332 
00333                 this.substrates(localizingProteins, this.compartmentIndexs_incorporated) = ...
00334                     this.substrates(localizingProteins, this.compartmentIndexs_incorporated) + ...
00335                     this.substrates(localizingProteins, this.compartmentIndexs_unincorporated);
00336                 this.substrates(localizingProteins, this.compartmentIndexs_unincorporated) = 0;
00337             end
