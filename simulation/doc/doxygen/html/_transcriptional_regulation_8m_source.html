00002 %
00003 % @wholeCellModelID Process_TranscriptionalRegulation
00004 % @name             Transcriptional Regulation
00005 % @description
00006 %   Biology
00007 %   ==============
00008 %   The the rate of transcription of each transcription unit is known to be
00009 %   regulated by proteins referred to as transcription factors which modulate
00010 %   the affinity of the RNA polymerase for each promoter. Transcription factors
00011 %   can both positively and negatively modulate RNA polymerase - promoter
00012 %   affinity. Transcription factors can stabilize RNA polymerase - promoter
00013 %   complexes by contributing additional negative free energy to the complex by
00014 %   providing an additional surface for the RNA polymerase binding.
00015 %   Transcription factors can destabilize RNA polymerase - promoter complexes
00016 %   for example by sterically blocking promoters and preventing the RNA
00017 %   polymerase from binding promoters.
00018 %
00019 %   This process simulates the binding of transcription factors to the promoters
00020 %   of each transcription unit (affinity), as well as the affect of the
00021 %   transcription factors on the recruiment of the RNA polymerase to each
00022 %   promoter (activity). The process was built using experimentally observed fold
00023 %   change expression effect of each transcription factor on each promoter.
00024 %
00025 %   Affinity
00026 %   ++++++++++++++
00027 %   Binds enzymes to promoters assuming:
00028 %   1) Transcription factors have high affinity for promoters
00029 %   2) Transcription factors bind promoters rapidly
00030 %   3) Transcription factors bind promoters stably over time
00031 %   4) Only 1 copy of a transcription factor can bind each promoter
00032 %   5) Transcription factors only compete within their species for
00033 %      promoters. That is for each transcription unit we assume each
00034 %      transcription factor species binds a distinct promoters region.
00035 %
00036 %   Consequently, at each time step we simulate that each free
00037 %   transcription factor binds randomly binds unoccupied promoters (no copy
00038 %   of that transcription factor is already bound to the promoter). Random
00039 %   transcription factor-promoter binding is weighted by the affinity of
00040 %   each transcription factor for each promoter.
00041 %
00042 %   Because transcription factor-promoter affinities are generally not
00043 %   experimentally observed, we base them on the transcription factor fold
00044 %   change activities.
00045 %
00046 %   Activity
00047 %   ++++++++++++++
00048 %   The effect of bound transcription factors on the recruitment of RNA
00049 %   polymerase and the expression of transcription units is simulated here
00050 %   and incorporated into the calculation of the RNA polymerase
00051 %   transcription unit promoter binding probabilities in the transcription
00052 %   process. Specifically, the wild-type average RNA polymerase
00053 %   transcription unit promoter binding probabilities are multiplied by the
00054 %   binding probability fold change effects simulated in this process.
00055 %
00056 %   The RNA polymerase binding probability fold change is simulated for
00057 %   each promoter as the product of the observed expression fold change
00058 %   effects of each bound transcription factor. When a promoter is bound by
00059 %   a single transcription factor, the net RNA polymerase binding
00060 %   probability fold change is the observed expression fold change of that
00061 %   transcription factor. When a promoter is bound by multiple
00062 %   transcription units, the net RNA polymerase binding probability fold
00063 %   change is given by the product of the individual fold change effects of
00064 %   the bound transcription factors.
00065 %
00066 %   Knowledge Base
00067 %   ==============
00068 %   The list of transcriptional regulatory relationships is maintained in the
00069 %   the knowledge base. As of 8/10/2010, it contained 31 such relationships
00070 %   between 5 transcription factors and 29 transcription units containing 37
00071 %   genes. The knowledge base was built from a variety of literature sources
00072 %   and databases including:
00073 %   - PUB_0096
00074 %   - PUB_0110
00075 %   - PUB_0112
00076 %   - PUB_0196
00077 %   - PUB_0418-20
00078 %   - PUB_0433-8
00079 %   - PUB_0505
00080 %
00081 %   Initialization
00082 %   =================
00083 %   Because we assume that transcription factors have high affinity for DNA and
00084 %   bind DNA stably, we initialize as many transcription factors as possible to
00085 %   the promoter-bound state. We randomly assign transcription factors to
00086 %   promoters using their relative affinities.
00087 %
00088 %   Simulation
00089 %   ==============
00090 %   For each kind of transcription factor, bind any free transcription factors
00091 %   to promoters that aren't already occupied by this kind of transcription
00092 %   factor. Choose the binding sites randomly, weighted by the transcription
00093 %   factor's affinity to them.

00359                 %  Stochastically pick transcription unit promoters elements to
00360                 %  bind from among free promoter elements weighted by the
00361                 %  transcription factors' affinities for the promoters. (Note:
00362                 %  each transcription factor is assumed to bind a distinct
00363                 %  promoter region near each transcription unit such that
00364                 %  transcription factors only compete within their species
00365                 %  for binding to promoters.)
00366                 
00367                 if this.enzymes(i) <= 0
00368                     continue;
00369                 end
00370                 
00371                 sites = find((this.tfIndexs == i) & accessible);
00372                 if isempty(sites)
00373                     continue;
00374                 end

00377                     this.tfPositionStrands(sites, :), i, this.enzymes(i), this.tfAffinities(sites), [], false, 1, false, []);
