00007 %
00008 % Public Variables

00016 % Local Variables:
00017 %   y1, y2, y3  ... three transformed solutions
00018 %
00019 % Formula used are given in Tuma, "Engineering Mathematics Handbook", p7
00020 %   (McGraw Hill, 1978).
00021 %   Step 0: If a is 0. use the quadratic formula to avoid dividing by 0.
00022 %   Step 1: Calculate p and q
00023 %           p = ( 3*c/a - (b/a)**2 ) / 3
00024 %           q = ( 2*(b/a)**3 - 9*b*c/a/a + 27*d/a ) / 27
00025 %   Step 2: Calculate discriminant D
00026 %           D = (p/3)**3 + (q/2)**2
00027 %   Step 3: Depending on the sign of D, we follow different strategy.
00028 %           If D<0, thre distinct real roots.
00029 %           If D=0, three real roots of which at least two are equal.
00030 %           If D>0, one real and two complex roots.
00031 %   Step 3a: For D>0 and D=0,
00032 %           Calculate u and v
00033 %           u = cubic_root(-q/2 + sqrt(D))
00034 %           v = cubic_root(-q/2 - sqrt(D))
00035 %           Find the three transformed roots
00036 %           y1 = u + v
00037 %           y2 = -(u+v)/2 + i (u-v)*sqrt(3)/2
00038 %           y3 = -(u+v)/2 - i (u-v)*sqrt(3)/2
00039 %   Step 3b Alternately, for D<0, a trigonometric formulation is more convenient
00040 %           y1 =  2 * sqrt(|p|/3) * cos(phi/3)
00041 %           y2 = -2 * sqrt(|p|/3) * cos((phi+pi)/3)
00042 %           y3 = -2 * sqrt(|p|/3) * cos((phi-pi)/3)
00043 %           where phi = acos(-q/2/sqrt(|p|**3/27))
00044 %                 pi  = 3.141592654...
00045 %   Step 4  Finally, find the three roots
00046 %           x = y - b/a/3

00048 
00049 %     pi = 3.141592654

00112     x(3) = y2r - y2i*1i;
