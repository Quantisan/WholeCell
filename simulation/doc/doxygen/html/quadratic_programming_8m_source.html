00001 function [x, objective, errorFlag, errorMsg] = quadraticProgramming(H, f, A, b, Aeq, beq, lb, ub, x0, options)
00002 % Provides a common interface to several quadratic programming solvers.
00003 % Checks each solver for errors.
00004 %
00005 % That is it solves the problem:
00006 %   min 0.5 * x'*H*x + f'*x
00007 %   subject to A*x<=b, Aeq*x=beq, lb<=x<=ub
00008 %
00009 % Requirements: at least one of the following quadratic programing solvers
00010 % - clp -- open source solve quadratic programming solver
00011 %   http://control.ee.ethz.ch/~joloef/mexclp.zip
00012 % - minq
00013 %   http://www.mat.univie.ac.at/~neum/software/minq/
00014 % - qpip -- open source solve quadratic programming solver
00015 %   http://sigpromu.org/quadprog/index.html
00016 % - Optimization Toolbox - has quadprog, MATLAB's quadratic programming
00017 %   solver. Note: quadprog is not very good. We suggest you use another
00018 %   solver instead.
00019 %   http://www.mathworks.com/products/optimization/
00020 % - Wolf
00021 %   http://www.mathworks.com/matlabcentral/fileexchange/27397-quadratic-programming-by-wolfs-method
00022 %
00023 % Author: Jonathan Karr
00024 % Affiliation: Covert Lab, Department of Bioengineering, Stanford University
00025 % Last updated: 1/17/2011
00026

00033 if exist('options', 'var') && isstruct(options) && isfield(options, 'solverOptions') && isfield(options.solverOptions, options.solver)
