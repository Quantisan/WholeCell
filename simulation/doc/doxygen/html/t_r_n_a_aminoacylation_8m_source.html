00001 %tRNA Aminoacylation
00002 %
00003 % @wholeCellModelID Process_tRNAAminoacylation
00004 % @name             tRNA Aminoacylation
00005 % @description
00006 %   Biology
00007 %   ===============
00008 %   In biological systems tRNAs serve as mediators between the ribosome and
00009 %   the amino acids which forms peptide polymers. In this process we
00010 %   simulate the conjugation of amino acids to the tRNAs which deliver them
00011 %   to the ribosome. Additionally we simulate the aminoacylation of the
00012 %   tmRNA which similarly delivers the amino acid alanine to stalled
00013 %   ribosomes.
00014 %
00015 %   Knowledge Base
00016 %   ===============
00017 %   As of 8/11/2010 the M. genitalium knowledge base contains 39 tRNA
00018 %   aminoacylation reactions involving
00019 %   - 37 aminoacylation reactions, each assuming a cost of 1 ATP per
00020 %     aminoacylation
00021 %   - 2 transfer reactions
00022 %   - 36 tRNAs and 1 tmRNA
00023 %   - 21 enzymes
00024 %   - 20 amino acid substrates and 10 additional substrates
00025 %
00026 %   The reaction kinetics stored in the knowledge base were compiled
00027 %   several sources including SABIO-RK [PUB_0100].
00028 %
00029 %   Note: Unlike knowledge base representation, for transfer reactions
00030 %   reactionStoichiometryMatrix doesn't include the already conjugated
00031 %   amino acid on the left-hand-side or the the ultimate conjugated amino
00032 %   acid on the right-hand-side.
00033 %
00034 %   Representation
00035 %   ===============
00036 %   substrates represents the counts of free metabolites available for tRNA
00037 %   aminoacylation. enzymes represents the counts of proteins available to
00038 %   catalyze tRNA aminoacylation reactions. freeRNAs and aminoacylatedRNAs
00039 %   represent the counts of free, unaminoacylated and aminocylated RNAs. We do
00040 %   not represent intermediate states in the aminoacylation of tRNAs.  The
00041 %   molecular weights of the unaminoacylated and aminocylated tRNAs are computed
00042 %   by the knowledge base RNA classes.
00043 %
00044 %   reactionStoichiometryMatrix, reactionModificationMatrix,
00045 %   reactionCatalysisMatrix, and enzymeBounds represent the tRNA aminoacylation
00046 %   reactions. reactionStoichiometryMatrix represents the free metabolites
00047 %   required for  each reaction. reactionModificationMatrix represents the tRNA
00048 %   aminoacylated by each reaction. reactionCatalysis represents the enzyme
00049 %   required to aminoacylate each tRNA. enzymeBounds represents the kcat of the
00050 %   catalyzing enzyme of each reaction. Note: reactionStoichiometryMatrix here
00051 %   is slightly modified from that computed by the super class: amino acids
00052 %   conjugated to tRNAs before and after transfer reactions have been removed
00053 %   from the left- and righ-hand-side of reactionStoichiometryMatrix.
00054 %
00055 %   Initialization
00056 %   ===============
00057 %   tRNAs are all initialized to the aminoacylated state.
00058 %
00059 %   Simulation
00060 %   ===============
00061 %   Uses greedy algorithm to simulate tRNA aminacylation (complexation of
00062 %   tRNAs with the specific amino acids that aminoacylate them).
00063 %   1. Deterministically activate tRNAs up to the minimum of free tRNAs and
00064 %      amino acids, proportional to free tRNAs
00065 %   2. Stoichastically activate residual tRNAs using residual amino acids
00066 %      with probabilities proportional to remaining free tRNAs
00067 %
00068 %      While(true)
00069 %        1. Calculate numbers of protein monomers that can be modified based on
00070 %           substrate, enzyme, and unmodified protein monomer availability and
00071 %           kinetics.
00072 %        2. Randomly select protein monomer to modify weighted by limits
00073 %           calculated in step (1).
00074 %        3. Update substrates, enzymes, unmodified protein monomers
00075 %        4. Repeat until insufficient resources to further modify protein
00076 %           monomers
00077 %      End
